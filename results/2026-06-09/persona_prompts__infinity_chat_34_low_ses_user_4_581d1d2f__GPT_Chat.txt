  - Solid build quality.

- Lenovo ThinkPad X280 or X390
  - More portable if you're carrying it to work.
  - Slightly smaller screen (12.5–13.3 inches).
  - Great for coding on the go.

Minimum specs I would recommend:
- 8th-generation Intel Core i5 or newer
- 8 GB RAM minimum (16 GB preferred)
- SSD storage (avoid hard drives)
- 1920×1080 (1080p) display if possible

- Python, VS Code, Jupyter Notebook, and beginner projects run comfortably on these systems.
- They can also handle introductory data analysis with pandas and NumPy.
- If you later move into machine learning, you can still learn the basics before needing more powerful hardware.

Models I would avoid:
- Chromebooks (unless you specifically want to learn Linux and development through workarounds)

- Intel Celeron or Pentium processors
