# Making a Zombie Survivor Game on Scratch

## Basic Setup

**Sprites you'll need:**
- Player character

## Core Game Elements

### Player Movement
```
when [green flag clicked]
forever
  if <key [w] pressed?> then
    change y by (5)
  end
  if <key [s] pressed?> then
    change y by (-5)
  end
  if <key [a] pressed?> then
    change x by (-5)
  end
  if <key [d] pressed?> then
    change x by (5)
  end
end
```

### Zombie AI (on Zombie sprite)
```
forever
  point towards [Player]
  move (2) steps
  if <touching [Player]?> then
    change [health] by (-10)
  end
end
```

### Shooting
```
when [space] pressed
  create clone of [Bullet]
```

**On Bullet sprite:**
```
when I start as a clone
  point towards (mouse-pointer)
  forever
    move (10) steps
    if <touching [Zombie]?> then
      delete this clone
    end
  end
```

## Important Variables to Create
- `health`
- `score`
- `wave number`
- `zombies remaining`

## Wave System
```
when [green flag clicked]
set [wave] to (1)
forever
  repeat (wave * 3)
    create clone of [Zombie]
    wait (1) seconds
  end
  wait until <[zombies remaining] = (0)>
  change [wave] by (1)
end
```
