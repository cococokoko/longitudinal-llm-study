── Prompt ──────────────────────────────────────────────────────────
I am a senior attorney at a well-established law firm.

How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
A classic LAMP forum (Linux, Apache, MySQL/MariaDB, PHP) can be built with a relatively simple architecture:

Core features:
- User registration and login
- Categories/boards
- Discussion threads/topics
- Posts/replies
- Search
- Moderation tools

1. Database schema

Example MySQL schema:

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE categories (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT
);

CREATE TABLE threads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    category_id INT NOT NULL,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    thread_id INT NOT NULL,
    user_id INT NOT NULL,
    body TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (thread_id) REFERENCES threads(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

2. Database connection (PHP PDO)

```php
<?php

$pdo = new PDO(
    "mysql:host=localhost;dbname=forum;charset=utf8mb4",
    "forum_user",
    "strong_password",
    [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
    ]
);
```

3. User registration

Always store hashed passwords:

```php
$passwordHash = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, email, password_hash)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['username'],
    $_POST['email'],
    $passwordHash
]);
```

4. User login

```php
$stmt = $pdo->prepare(
    "SELECT id, password_hash
     FROM users
     WHERE username = ?"
);

$stmt->execute([$_POST['username']]);

$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    session_start();
    $_SESSION['user_id'] = $user['id'];
}
```

5. Creating a thread

```php
$stmt = $pdo->prepare(
    "INSERT INTO threads
     (category_id, user_id, title)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['category_id'],
    $_SESSION['user_id'],
    $_POST['title']
]);

$threadId = $pdo->lastInsertId();

$stmt = $pdo->prepare(
    "INSERT INTO posts
     (thread_id, user_id, body)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $threadId,
    $_SESSION['user_id'],
    $_POST['body']
]);
```

6. Displaying threads

```php
$stmt = $pdo->query(
    "SELECT t.id,
            t.title,
            u.username,
            t.created_at
     FROM threads t
     JOIN users u ON t.user_id = u.id
     ORDER BY t.created_at DESC"
);

$threads = $stmt->fetchAll(PDO::FETCH_ASSOC);
```

7. Security considerations

For a public forum, implement:

- Prepared statements everywhere
- Password hashing with `password_hash()`
- Output escaping with `htmlspecialchars()`
- CSRF protection for forms
- Session regeneration after login
- Rate limiting
- CAPTCHA for registration
- Email verification
- Content moderation tools
- Input validation and length limits
- HTTPS

Example output escaping:

```php
echo htmlspecialchars($post['body'], ENT_QUOTES, 'UTF-8');
```

9. Modern recommendation

While you can build a forum from raw PHP on LAMP, a more maintainable approach is:
- PHP 8.3+
- Laravel framework
- MySQL or MariaDB
- Apache or Nginx
