# Running Dolphin Emulator on Microsoft Surface with Touch Controls

## What You'll Need
- Microsoft Surface (with Windows)
- Dolphin Emulator installed
- Wii game ROMs (that you legally own)
- Patience with touch controls

---

## Step 1: Download & Install Dolphin

1. Go to **dolphin-emu.org** in Edge
2. Tap **Download** → choose the latest version
3. Tap the downloaded file to install
4. Follow the installation prompts

---

## Step 2: Configure Touch Input for Dolphin

Dolphin doesn't natively support touchscreen well, so here are your options:

### Option A: Use the Touchpad as a Mouse
- The touchpad works as a standard mouse pointer
- You can map mouse clicks to Wii controller buttons in Dolphin settings

### Option B: Map Keyboard to Controller
1. Open Dolphin
2. Go to **Controllers** at the top
3. Select **Emulated Wii Remote**
4. Map buttons to keyboard keys
5. Use the **on-screen keyboard** to press those keys during gameplay

---

## Step 3: Configure the Wiimote

1. Tap **Controllers** in Dolphin toolbar
2. Select **Wii Remote 1** → change to **Emulate the Wii's Bluetooth Adapter** or **Emulated Wii Remote**
3. Tap **Configure**
4. Map these essential buttons:
   - **A Button** → Enter or a touch-friendly key
   - **B Button** → Backspace or another key
   - **D-Pad** → Arrow keys
   - **Home** → H key
   - **+/-** → + and - keys

---

## Practical Tips for Touch Use

- **Pin the on-screen keyboard** so it stays visible during gameplay
- Use **tablet mode** in Windows for easier touch navigation
- Consider a **Bluetooth controller** - this would be MUCH easier than touch
- Some games work better than others with limited controls

---

## Honest Recommendation

Touch + on-screen keyboard is **very awkward** for most Wii games. The best affordable options:

| Option | Cost | Ease |
|--------|------|------|
| Xbox controller via USB | ~$25 used | Very Easy |
| Bluetooth gamepad | ~$15-30 | Easy |
| On-screen keyboard | Free | Difficult |
