# Making a Zombie Survivor Game on Scratch

Here's a practical guide to building a solid zombie survivor game!

---

## Core Components You'll Need

### 1. **Sprites to Create**
- Player character
- Zombie (enemy)
- Bullet/projectile
- Health bar or hearts
- Game Over screen

---

## Basic Setup

### Player Movement
```
when [green flag clicked]
forever
  if <key [w] pressed?> then
    change y by (5)
  end
  if <key [s] pressed?> then
    change y by (-5)
  end
  if <key [a] pressed?> then
    change x by (-5)
  end
  if <key [d] pressed?> then
    change x by (5)
  end
end
```

### Player Shooting
```
when [space v] key pressed
  create clone of [Bullet v]
```

**In the Bullet sprite:**
```
when I start as a clone
  point in direction (direction of [Player v])
  forever
    move (10) steps
    if <touching [Zombie v]?> then
      delete this clone
    end
    if <touching [edge v]?> then
      delete this clone
    end
  end
```

---

### Zombie Behavior
```
when [green flag clicked]
forever
  point towards [Player v]
  move (2) steps
  if <touching [Player v]?> then
    change [Health v] by (-1)
    wait (0.5) seconds
  end
end
```

---

### Zombie Spawning
```
when [green flag clicked]
forever
  wait (3) seconds
  create clone of [Zombie v]

when I start as a clone
  go to x: (pick random -240 to 240) y: (pick random -180 to 180)
  show
```

## Health System
```
when [green flag clicked]
set [Health v] to (3)
forever
  if <(Health) < (1)> then
    broadcast [Game Over v]
  end
end
```
