# Running Dolphin Emulator on Microsoft Surface

Here's a practical guide tailored to your Surface's input methods.

---

## Setup First

### Download & Install Dolphin
1. Go to **dolphin-emu.org**
2. Download the latest stable release (Windows x64)
3. Run the installer - standard Next/Next/Finish process

---

## Configuring Input for Surface Touchpad

### Touchpad as Controller
1. Open Dolphin → **Controllers** (top menu)
2. Select **Emulated Wii Remote** for Port 1
3. Click **Configure**
4. Map touchpad gestures/clicks to Wii buttons

### Practical Touchpad Mappings
| Surface Touchpad | Wii Function |
|---|---|
| Left click | A button |
| Right click | B button |
| Two-finger tap | Menu/Home |
| Swipe | Motion controls |

---

## On-Screen Keyboard Tips
- Enable via **taskbar system tray** or search "Touch Keyboard"
- Useful for **entering game settings** rather than gameplay
- Consider keeping it docked at bottom during setup

---

## Honest Recommendation
For actual gameplay, the touchpad and on-screen keyboard will be **genuinely frustrating**. I'd strongly suggest:
- A **Bluetooth controller** (Xbox controller pairs easily)
- A **USB gamepad**
