── Prompt ──────────────────────────────────────────────────────────
How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
Building an online forum with the LAMP stack (Linux, Apache, MySQL/MariaDB, PHP) involves creating user accounts, discussion categories, threads, posts, and moderation features. Here's a practical roadmap.

1. Set up the LAMP environment

Install:
- Linux (Ubuntu is common)
- Apache
- MySQL or MariaDB
- PHP 8.x

Example (Ubuntu):

```bash
sudo apt update
sudo apt install apache2 mariadb-server php php-mysql
```

Create a database:

```sql
CREATE DATABASE forum_db;
CREATE USER 'forum_user'@'localhost' IDENTIFIED BY 'strong_password';
GRANT ALL PRIVILEGES ON forum_db.* TO 'forum_user'@'localhost';
FLUSH PRIVILEGES;
```

2. Design the database

A minimal schema:

Users:

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Categories:

```sql
CREATE TABLE categories (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT
);
```

Threads:

```sql
CREATE TABLE threads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    category_id INT NOT NULL,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

Posts:

```sql
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    thread_id INT NOT NULL,
    user_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (thread_id) REFERENCES threads(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

4. Create a database connection

config/database.php

```php
<?php

$host = "localhost";
$dbname = "forum_db";
$user = "forum_user";
$pass = "strong_password";

try {
    $pdo = new PDO(
        "mysql:host=$host;dbname=$dbname;charset=utf8mb4",
        $user,
        $pass
    );

    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

} catch(PDOException $e) {
    die("Connection failed");
}
```

5. Build user registration

Use password hashing.

```php
$passwordHash = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, email, password_hash)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['username'],
    $_POST['email'],
    $passwordHash
]);
```

6. Build login

```php
$stmt = $pdo->prepare(
    "SELECT * FROM users WHERE username = ?"
);

$stmt->execute([$_POST['username']]);

$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    session_start();
    $_SESSION['user_id'] = $user['id'];
}
```
