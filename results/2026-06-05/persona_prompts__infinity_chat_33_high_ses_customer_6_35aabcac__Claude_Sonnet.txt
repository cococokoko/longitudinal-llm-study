# Running Dolphin Emulator on Microsoft Surface

Here's a practical guide for your setup:

## Installation First

1. Go to **dolphin-emu.org**
2. Download the latest stable release (Windows x64)
3. Run the installer - no admin rights typically needed

## Touchpad Configuration

**Basic Navigation:**
- **Single tap** = left click
- **Two-finger tap** = right click
- **Two-finger scroll** = navigate menus
- **Pinch** = zoom in Dolphin's interface

**In Dolphin Settings (Options → Controller Settings):**
- Set controllers to **"Emulated Wii Remote"**
- Map touchpad gestures under the input configuration

## On-Screen Keyboard Tips

- **Windows key + Ctrl + O** opens the built-in touch keyboard
- Pin it to your taskbar for quick access
- Use it primarily for text fields in Dolphin menus

## Honest Practical Considerations

I want to be straightforward with you:

- **Touchpad + on-screen keyboard is genuinely difficult** for most Wii games
- Many games require motion controls or analog sticks
- You'll likely want to connect a **Bluetooth controller** (Xbox or actual Wiimote) for real gameplay
- A **USB controller** is the simplest solution

## Recommended Quick Fix
Connect any USB gamepad - Dolphin recognizes them instantly and makes the experience significantly better.
