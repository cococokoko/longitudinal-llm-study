How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
A LAMP forum (Linux, Apache, MySQL/MariaDB, PHP) can be built with a few core components:

- User accounts (register, login, logout)
- Discussion categories/boards
- Topics (threads)
- Posts (replies)
- Database storage
- Session-based authentication

Basic architecture

- Apache serves PHP pages.
- PHP handles requests and database queries.
- MySQL stores users, topics, and posts.
- Sessions keep users logged in.

Example database schema

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    topic_id INT NOT NULL,
    user_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

Database connection (db.php)

```php
<?php

$host = "localhost";
$dbname = "forum";
$user = "forum_user";
$pass = "your_password";

$pdo = new PDO(
    "mysql:host=$host;dbname=$dbname;charset=utf8mb4",
    $user,
    $pass,
    [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
    ]
);
```

User registration

```php
<?php
require "db.php";

$username = $_POST['username'];
$password = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, password_hash)
     VALUES (?, ?)"
);

$stmt->execute([$username, $password]);

echo "Registration successful";
```

User login

```php
<?php
session_start();
require "db.php";

$stmt = $pdo->prepare(
    "SELECT * FROM users WHERE username = ?"
);

$stmt->execute([$_POST['username']]);
$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    $_SESSION['user_id'] = $user['id'];
    echo "Logged in";
} else {
    echo "Invalid credentials";
}
```

Display a topic and its posts

```php
<?php
require "db.php";

$topicId = (int)$_GET['id'];

$stmt = $pdo->prepare(
    "SELECT title FROM topics WHERE id = ?"
);
$stmt->execute([$topicId]);
$topic = $stmt->fetch();

echo "<h1>" . htmlspecialchars($topic['title']) . "</h1>";

$stmt = $pdo->prepare(
    "SELECT p.content, u.username
     FROM posts p
     JOIN users u ON p.user_id = u.id
     WHERE p.topic_id = ?
     ORDER BY p.created_at"
);

$stmt->execute([$topicId]);

foreach ($stmt as $post) {
    echo "<p><strong>" .
         htmlspecialchars($post['username']) .
         ":</strong> " .
         nl2br(htmlspecialchars($post['content'])) .
         "</p>";
}
```

Important security practices

- Use prepared statements (PDO or MySQLi).
- Hash passwords with `password_hash()`.
- Verify with `password_verify()`.
- Escape output using `htmlspecialchars()`.
- Add CSRF protection for forms.
- Validate and sanitize all user input.
- Use HTTPS.
- Restrict permissions on uploaded files if you support attachments.

If you're building a production forum, you may also consider using an existing PHP forum package such as phpBB, Flarum, or MyBB rather than building everything from scratch. They already include moderation tools, permissions, notifications, and security features.
