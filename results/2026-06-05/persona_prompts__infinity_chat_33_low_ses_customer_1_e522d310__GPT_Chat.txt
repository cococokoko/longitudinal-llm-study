You can run Dolphin on a Microsoft Surface using only the touchpad and the Windows on-screen keyboard, but you'll need to configure the controls first.

For Wii games, the easiest approach is usually to emulate a Wii Remote with your keyboard and touchpad.

1. Open Dolphin
- Launch Dolphin.
- Click Controllers.

2. Configure the Wii Remote
- Under "Wiimotes," set Wii Remote 1 to:
  - Emulated Wii Remote
- Click Configure.

3. Map buttons to the on-screen keyboard
With the Windows on-screen keyboard open (Win + Ctrl + O on newer Windows versions, or search for "On-Screen Keyboard"), click each Wii button field in Dolphin and assign keys such as:
- A = Z
- B = X
- 1 = 1
- 2 = 2
- Plus = Enter
- Minus = Backspace
- Home = H
- D-Pad = Arrow Keys

You can choose any keys that are comfortable.

4. Use the touchpad for pointing
Many Wii games require the Wii Remote pointer.

In the Wii Remote configuration:
- Locate Point.
- Map:
  - Up = Mouse Y-
  - Down = Mouse Y+
  - Left = Mouse X-
  - Right = Mouse X+

This allows the Surface touchpad to control the Wii pointer.

5. Map touchpad clicks
For games that use the Wii Remote trigger:
- B = Left Mouse Button

You can then click the touchpad to simulate holding the Wii Remote trigger.

6. Motion controls (if needed)
Some Wii games require shaking or tilting.

In the Motion Input section, assign keyboard keys:
- Shake X = Q
- Shake Y = W
- Shake Z = E

Pressing those keys on the on-screen keyboard will trigger shake actions.

7. Save the profile
- Click Save.
- Give the profile a name such as "Surface Touchpad Wii."
