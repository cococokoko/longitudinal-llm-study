## Linear Graph Example

A **path graph** is a classic example of a linear graph in graph theory.

### Example: P₄ (Path Graph with 4 vertices)

```
A --- B --- C --- D
```

### Properties
- **Vertices:** {A, B, C, D}
- **Edges:** {AB, BC, CD}
- **Degree sequence:** A and D have degree 1 (endpoints), B and C have degree 2

### Key Characteristics of a Linear Graph
- Each vertex is connected in a sequential, line-like structure
- Exactly **two vertices of degree 1** (the endpoints)
- All internal vertices have **degree 2**
- It is **acyclic** (no cycles)
- It has **n - 1 edges** for n vertices

### Formal Definition
A path graph **Pₙ** on *n* vertices can be defined as:
- V = {v₁, v₂, ..., vₙ}
- E = {v₁v₂, v₂v₃, ..., vₙ₋₁vₙ}

This is arguably the simplest non-trivial graph structure and serves as a foundational example in graph theory courses.
