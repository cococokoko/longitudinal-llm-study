Give a numerical example to illustrate the concept of partial derivative.

── Response ────────────────────────────────────────────────────────
A partial derivative measures how a function changes with respect to one variable while holding the other variables constant.

Example:

Consider the function

f(x, y) = x²y + 3xy

Suppose we want the partial derivative with respect to x.

1. Treat y as a constant and differentiate with respect to x:

∂f/∂x = 2xy + 3y

2. Evaluate it at the point (x, y) = (2, 4):

∂f/∂x(2,4) = 2(2)(4) + 3(4)
            = 16 + 12
            = 28

Interpretation:
- Near the point (2,4), if x increases by a small amount while y stays fixed at 4, the function increases by approximately 28 times that small change.
- For example, increasing x from 2 to 2.01 (Δx = 0.01) would increase f by roughly:
  Δf ≈ 28 × 0.01 = 0.28

Similarly, for the partial derivative with respect to y:

1. Treat x as a constant:

∂f/∂y = x² + 3x

2. Evaluate at (2,4):

∂f/∂y(2,4) = 2² + 3(2)
            = 4 + 6
            = 10

Interpretation:
- Near (2,4), if y increases by a small amount while x remains fixed at 2, the function increases by approximately 10 times that small change.

So at the point (2,4):
- ∂f/∂x = 28
- ∂f/∂y = 10

These numbers quantify the sensitivity of the function to changes in x and y independently.
