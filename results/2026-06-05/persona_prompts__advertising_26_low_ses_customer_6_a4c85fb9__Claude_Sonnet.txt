# Data Security Best Practices for Small Businesses

Here are key practices organized by priority:

## Foundation Security
- **Use strong, unique passwords** and a password manager
- **Enable multi-factor authentication (MFA)** on all accounts
- **Keep software updated** - patches fix known vulnerabilities
- **Install reputable antivirus/antimalware** software

## Data Protection
- **Regular backups** using the 3-2-1 rule (3 copies, 2 media types, 1 offsite)
- **Encrypt sensitive data** both stored and in transit
- **Limit data collection** - only keep what you actually need
- **Secure disposal** of old devices and documents

## Network Security
- Use a **business-grade firewall**
- Separate **guest WiFi** from business networks
- Consider a **VPN** for remote access

## Employee Practices
- **Train staff regularly** on phishing and social engineering
- Apply **least privilege access** - employees only access what they need
- Have a clear **acceptable use policy**

## Planning
- Create an **incident response plan** before you need one
- Know your **legal obligations** regarding customer data
- Consider **cyber liability insurance**

## Quick Wins
- Audit who has access to what
- Enable automatic updates
- Use cloud services with strong security reputations
