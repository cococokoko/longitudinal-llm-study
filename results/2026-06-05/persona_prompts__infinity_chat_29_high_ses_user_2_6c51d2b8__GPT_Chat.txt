── Prompt ──────────────────────────────────────────────────────────
I am a senior attorney at a well-established law firm.

How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
A LAMP forum (Linux, Apache, MySQL/MariaDB, PHP) can be built with a fairly standard web application architecture:

1. Set up the LAMP stack

On Ubuntu:

```bash
sudo apt update
sudo apt install apache2 mysql-server php php-mysql
```

Verify:

```bash
apache2 -v
php -v
mysql --version
```

2. Create a database

Example schema:

```sql
CREATE DATABASE forum_db;

USE forum_db;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE threads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    thread_id INT NOT NULL,
    user_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (thread_id) REFERENCES threads(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

3. Create a database connection

`config.php`

```php
<?php

$host = "localhost";
$dbname = "forum_db";
$user = "forum_user";
$pass = "strong_password";

try {
    $pdo = new PDO(
        "mysql:host=$host;dbname=$dbname;charset=utf8mb4",
        $user,
        $pass
    );

    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

} catch (PDOException $e) {
    die("Connection failed");
}
```

4. User registration

Example registration logic:

```php
<?php

require 'config.php';

$username = $_POST['username'];
$password = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, password_hash)
     VALUES (?, ?)"
);

$stmt->execute([$username, $password]);
```

5. User login

```php
<?php

session_start();
require 'config.php';

$stmt = $pdo->prepare(
    "SELECT * FROM users WHERE username = ?"
);

$stmt->execute([$_POST['username']]);

$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    $_SESSION['user_id'] = $user['id'];
}
```

6. Create a discussion thread

```php
<?php

session_start();
require 'config.php';

$stmt = $pdo->prepare(
    "INSERT INTO threads (user_id, title)
     VALUES (?, ?)"
);

$stmt->execute([
    $_SESSION['user_id'],
    $_POST['title']
]);
```

7. Add a post

```php
<?php

session_start();
require 'config.php';

$stmt = $pdo->prepare(
    "INSERT INTO posts (thread_id, user_id, content)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['thread_id'],
    $_SESSION['user_id'],
    $_POST['content']
]);
```

8. Display threads

```php
<?php

require 'config.php';

$stmt = $pdo->query(
    "SELECT t.id, t.title, u.username
     FROM threads t
     JOIN users u ON t.user_id = u.id
     ORDER BY t.created_at DESC"
);

while ($row = $stmt->fetch()) {
    echo "<a href='thread.php?id={$row['id']}'>";
    echo htmlspecialchars($row['title']);
    echo "</a> by ";
    echo htmlspecialchars($row['username']);
    echo "<br>";
}
```

9. Security considerations

For a public forum, implement:

- Prepared statements everywhere (to prevent SQL injection)
- Password hashing with `password_hash()`
- Output escaping with `htmlspecialchars()`
- CSRF protection on forms
- Session security
- Rate limiting and spam protection
- User roles (admin/moderator/member)
- Content reporting and moderation tools
- HTTPS/TLS
