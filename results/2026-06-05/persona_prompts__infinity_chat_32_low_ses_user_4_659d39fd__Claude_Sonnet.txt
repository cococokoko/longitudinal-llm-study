# Making a Zombie Survivor Game on Scratch

Here's a straightforward guide to building a basic zombie survival game!

---

## Basic Game Concept
- Player moves around avoiding/fighting zombies
- Zombies chase the player
- Player has health points
- Survive as long as possible

---

## Step 1: Set Up Your Project

- Go to **scratch.mit.edu**
- Click **Create**
- Set a dark background (graveyard, city, etc.)

---

## Step 2: Create Your Sprites

**You'll need:**
- Player sprite
- Zombie sprite
- Health bar or hearts
- Bullet/weapon sprite (optional)

---

## Step 3: Player Movement Code

On your **Player sprite:**

```
When Green Flag clicked
forever
  if key [W] pressed then
    change y by 3
  if key [S] pressed then
    change y by -3
  if key [A] pressed then
    change x by -3
  if key [D] pressed then
    change x by -3
```

---

## Step 4: Zombie AI (Chasing Player)

On your **Zombie sprite:**

```
When Green Flag clicked
forever
  point toward [Player]
  move 2 steps
```

---

## Step 5: Zombie Damages Player

```
When Green Flag clicked
forever
  if touching [Player] then
    change [Health] by -1
    wait 1 second
```

---

## Step 6: Health System

```
When Green Flag clicked
set [Health] to 100
forever
  if [Health] < 1 then
    broadcast [Game Over]
    stop all
```

---

## Step 7: Spawning Multiple Zombies

- **Clone** your zombie sprite
- Use this on the zombie:

```
When Green Flag clicked
forever
  create clone of [myself]
  wait 3 seconds

When I start as a clone
go to x: (pick random -200 to 200) y: (pick random -150 to 150)
forever
  point toward [Player]
  move 2 steps
```

---

## Step 8: Add a Weapon (Optional)

**On click, shoot a bullet:**

```
When this sprite clicked
  create clone of [Bullet]

When I start as a clone
point toward [mouse pointer]
forever
  move 10 steps
  if touching [Zombie] then
    change [ZombieHealth] by -10
    delete this clone
  if touching edge then
    delete this clone
```

---

## Tips to Make It Better

| Feature | How |
|---|---|
| Score | Add to score variable each second survived |
| Speed increase | Make zombies faster over time |
| Sound effects | Add zombie/hurt sounds |
| Game Over screen | Show score when player dies |
| Multiple lives | Start with 3 hearts |

---

## Helpful Scratch Features to Use
- **Variables** - health, score, wave number
- **Clones** - spawning multiple zombies
- **Broadcasts** - triggering game over
- **My Blocks** - organizing your code
