# Making a Zombie Survivor Game on Scratch

Here's a breakdown of how to build it!

## Basic Setup

**Sprites you'll need:**
- Player character
- Zombie (enemy)
- Bullets/projectiles
- Health bar or hearts
- Background/arena

---

## Core Game Elements

### Player Movement
```
when [green flag clicked]
forever
  if <key [w] pressed?> then
    change y by (5)
  end
  if <key [s] pressed?> then
    change y by (-5)
  end
  if <key [a] pressed?> then
    change x by (-5)
  end
  if <key [d] pressed?> then
    change x by (5)
  end
end
```

### Zombie Following Player
```
when [green flag clicked]
forever
  point towards [Player]
  move (2) steps
end
```

### Shooting Mechanic
```
when [space] key pressed
create clone of [Bullet]
```

**Bullet sprite script:**
```
when I start as a clone
point towards (mouse-pointer)
forever
  move (10) steps
  if <touching [Zombie]?> then
    delete this clone
  end
end
```

---

## Important Variables to Create
- **Health** (start at 100)
- **Score**
- **Wave Number**
- **Zombie Speed**

---

## Zombie Damage Script
```
when [green flag clicked]
forever
  if <touching [Player]?> then
    change [Health] by (-1)
    wait (0.5) seconds
  end
end
```

---

## Wave System
```
when [green flag clicked]
set [Wave] to (1)
forever
  wait (30) seconds
  change [Wave] by (1)
  // spawn more zombies each wave
end
```

---

## Tips for Your Game
- Use **clones** for multiple zombies
- Add **sound effects** for atmosphere
- Create a **game over screen** when health hits 0
- Add **walls** using color detection
- Use **costumes** for animation
