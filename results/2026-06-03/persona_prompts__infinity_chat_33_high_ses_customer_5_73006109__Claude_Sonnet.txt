# Running Dolphin Emulator on Microsoft Surface

Here's a practical guide for your setup:

## Installation First

1. Download Dolphin from **dolphin-emu.org** (official site only)
2. Run the installer and follow standard setup
3. Add your legally-owned Wii game files (ISO/WBFS format)

---

## Touchpad Configuration

**In Dolphin's Controller Settings:**
- Go to **Options → Controller Settings**
- Set GameCube/Wii controllers to **"Emulated"**
- Click **Configure** to map inputs

**Touchpad Mapping Tips:**
- Map touchpad clicks to buttons (A, B)
- Use touchpad gestures for basic navigation
- Consider enabling **touchpad zones** for different button areas

---

## On-Screen Keyboard Considerations

The on-screen keyboard works best for:
- Menu navigation
- Text entry in Dolphin settings
- **Not ideal** for actual gameplay

---

## Honest Recommendation

For actual gameplay, this input combination will be **quite frustrating**. I'd suggest:
- A **Bluetooth controller** (Xbox, PlayStation, or actual Wiimote)
- A **USB controller**
- Even a basic **USB keyboard** for better key mapping
