Give an example of a linear graph in graph theory.

── Response ────────────────────────────────────────────────────────
In graph theory, a “linear graph” is usually just a path graph — a set of vertices connected in a straight line.

Example:

Vertices: {A, B, C, D, E}  
Edges: {A–B, B–C, C–D, D–E}

You can picture it like this:

A — B — C — D — E

Each node (except the ends) connects to exactly two others, forming a simple line. This is often called a path graph P5 (because it has 5 vertices).
