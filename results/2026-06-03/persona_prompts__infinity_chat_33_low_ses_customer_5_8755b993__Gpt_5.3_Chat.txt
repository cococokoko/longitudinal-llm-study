You can run Dolphin pretty comfortably on a Surface with just the touchpad and on-screen keyboard—it just takes a bit of setup. Here’s a practical way to make it usable.

First, install and launch Dolphin normally. Once it’s open, tap “Controllers.”

Set up the Wii Remote:
- Under “Wii Remotes,” set “Emulated Wii Remote” for Player 1.
- Click “Configure.”

Now map everything to things you can actually do with a touchpad:

Core buttons:
- A button → left-click (or a keyboard key like Space if you prefer)
- B trigger → right-click
- 1 / 2 buttons → map to easy keys like Q and E
- Plus / Minus → Enter and Backspace or similar

D-pad:
- Map to arrow keys (these work fine with the on-screen keyboard)

Motion controls (this is the tricky part):
- Pointer (for aiming): enable “Mouse” input for IR. This lets you use the touchpad like a Wii pointer.
- Tilt / shake:
  - Assign “shake” to a key (like Shift)
  - For tilt, map to keys (WASD works well)

You won’t get true motion control, but most games become playable with these mappings.

Touchpad tips:
- Use single-finger movement for pointer control
- Two-finger tap for right-click (useful if you mapped B there)
- Tap-to-click is easier than physically clicking

On-screen keyboard:
- Open it from the taskbar when needed
- For smoother gameplay, try to rely mostly on mapped mouse/touchpad inputs so you’re not constantly popping it up

Important Dolphin settings:
- In “Config” > “General,” enable “Dual Core” for better performance
- In “Graphics,” use Vulkan or Direct3D 11 depending on what runs smoother on your Surface

Optional but helpful:
- If your Surface supports touchscreen input, Dolphin can sometimes treat touch as pointer movement—this can feel more natural than the touchpad for Wii IR

Reality check:
Some Wii games (especially ones that rely heavily on motion like Wii Sports) won’t feel great without a real controller. But slower-paced or pointer-light games (like menus, RPGs, or some platformers) can work surprisingly well.
