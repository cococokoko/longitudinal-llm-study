# Data Security Best Practices for Small Businesses

Here are the key areas to focus on, organized for practical implementation:

## Essential Foundations
- **Use strong, unique passwords** for all accounts
- **Enable two-factor authentication (2FA)** wherever possible
- **Keep software updated** - patches fix security vulnerabilities
- **Install reputable antivirus/antimalware** software

## Data Protection
- **Back up data regularly** using the 3-2-1 rule
  - 3 copies, on 2 different media, 1 offsite/cloud
- **Encrypt sensitive files** and customer data
- **Limit who can access** what information (least privilege principle)

## Network Security
- Use a **secure, password-protected WiFi**
- Consider a **VPN** for remote work
- Separate guest networks from business networks

## Human Factors
- **Train employees** to recognize phishing emails
- Create a **clear security policy** in writing
- Have a **response plan** if a breach occurs

## Practical Starting Points
| Priority | Action | Cost |
|----------|--------|------|
| High | Enable 2FA | Free |
| High | Automate backups | Low |
| Medium | Password manager | Low |
| Medium | Employee training | Low-Free |

## Worth Knowing
- Many **free resources** exist through CISA.gov
- **Cyber insurance** is increasingly affordable for small businesses
