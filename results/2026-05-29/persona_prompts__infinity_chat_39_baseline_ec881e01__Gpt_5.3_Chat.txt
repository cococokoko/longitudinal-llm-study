13. Time-Saving Tools and Apps

The goal isn’t to use more tools. It’s to remove friction. The best tools are quiet, fast, and reduce decisions.

Task capture  
Use one place for everything. Examples: Todoist, Apple Reminders, Microsoft To Do.  
Quick entry matters more than features. If it takes more than a few seconds, you won’t use it.

Calendar control  
Your calendar is your boundary. Examples: Google Calendar, Fantastical.  
Block time for focused work, not just meetings. Treat those blocks as non-negotiable.

Automation  
Automate repeat actions. Examples: Zapier, Make, IFTTT.  
Simple use cases: save email attachments to cloud, auto-log expenses, sync notes across apps.

Notes and reference  
Store ideas where they’re easy to find. Examples: Notion, Apple Notes, Obsidian.  
Don’t over-organize. Use search. Tag lightly, or not at all.

Password management  
Remove the need to remember. Examples: 1Password, Bitwarden.  
This saves time and reduces mental load every day.

Templates  
Reuse common work. Email replies, reports, checklists.  
A saved template can turn a 10-minute task into 1 minute.

Focus tools  
Reduce distractions. Examples: Forest, Freedom, Cold Turkey.  
Block what you don’t need. Silence is a tool.

File storage  
Keep files accessible everywhere. Examples: Google Drive, Dropbox, iCloud.  
Avoid duplicate systems. One source of truth.

Quick text expansion  
Type less. Examples: TextExpander, built-in keyboard shortcuts.  
Short codes for long phrases, addresses, or replies.

Meeting reduction  
Use async tools instead. Examples: Loom for video updates, Slack or Teams for quick decisions.  
Fewer meetings = more real work.

Rule of thumb  
If a tool doesn’t save time within a week, remove it. The simplest stack wins.
