── Prompt ──────────────────────────────────────────────────────────
Can you write me a series of three questions that can determine if something is morally right or wrong?

── Response ────────────────────────────────────────────────────────
Here’s a simple three-question test you can run anything through:

1. Does this cause unnecessary harm or violate someone’s basic rights?
2. Would it still seem acceptable if everyone did it in similar circumstances?
3. Would I be okay if this action were done to me (or someone I care about)?

If something fails one or more of these, it’s a strong sign it’s morally questionable. If it passes all three, it’s at least on solid ethical ground.
