# Remote Team Collaboration Tools for Research Groups

Here's a practical breakdown by function:

## Communication
- **Slack** - Organized channels by project, integrates with many research tools
- **Microsoft Teams** - Good if your institution uses Office 365
- **Zoom/Teams** - Video meetings, lab meetings, journal clubs

## Project & Task Management
- **Asana or Trello** - Task tracking, deadlines, accountability
- **Notion** - Flexible workspace combining notes, tasks, and databases
- **Monday.com** - Visual project timelines

## Document Collaboration
- **Google Workspace** - Real-time document/spreadsheet editing
- **Overleaf** - Essential for collaborative manuscript writing (LaTeX)
- **Microsoft SharePoint** - Institutional document management

## Research-Specific Tools
- **GitHub** - Code version control for computational work
- **OSF (Open Science Framework)** - Project management + data sharing
- **Benchling** - Lab notebook and protocol management
- **Zotero/Mendeley** - Shared reference libraries

## Data & Analysis
- **Dropbox/Box** - File sharing (check HIPAA compliance needs)
- **REDCap** - Clinical data management

## Practical Recommendations
- **Start simple** - avoid tool overload
- Establish clear **communication norms** (response time expectations)
- Regular **structured check-ins** matter more than the tools themselves
