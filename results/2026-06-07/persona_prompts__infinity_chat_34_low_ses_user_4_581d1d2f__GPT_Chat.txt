── Response ────────────────────────────────────────────────────────
For learning Python, you do not need a powerful laptop. A reliable business-class laptop that was originally sold to companies is often the best value on the secondhand market because they're durable, repairable, and usually have good keyboards.

Good secondhand options:

- Lenovo ThinkPad T480 (2018)
  - One of the most recommended used laptops.
  - Excellent keyboard.
  - Easy to upgrade RAM and storage.
  - Supports Windows 11.
  - Look for:
    - Intel Core i5-8250U or i5-8350U
    - 16GB RAM if possible (8GB minimum)
    - 256GB+ SSD

- Lenovo ThinkPad T490
  - Slightly newer and lighter than the T480.
  - Good battery life.
  - Often available at reasonable prices.

- Dell Latitude 5490, 5400, or 5410
  - Business-class reliability.
  - Good Linux compatibility if you decide to experiment with Linux later.
  - Easy to find from corporate refurbishers.

- HP EliteBook 840 G5 or G6
  - Solid build quality.
  - Good screen options.
  - Usually competitively priced used.

- Lenovo ThinkPad X280 or X390
  - Smaller 12.5–13-inch laptops.
  - Great if you commute or carry it to work frequently.
  - Less upgradeable than the T-series but very portable.

If you're on a tighter budget:

- ThinkPad T470
- Dell Latitude 7480
- HP EliteBook 840 G4

These can often be found for $150–250 USD and are still perfectly capable of running:
- Python
- VS Code
- Jupyter Notebooks
- Git
- Basic web development tools

Specifications I'd recommend in 2026:
- CPU: Intel 8th-gen Core i5 or newer
- RAM: 16GB preferred, 8GB minimum
- Storage: 256GB SSD minimum
- Screen: 1920×1080 (Full HD) if possible

I'd avoid:
- Chromebooks (unless you specifically want to learn in a Linux environment)
- Very old laptops with spinning hard drives
- Consumer models with 4GB RAM
- Anything older than Intel 8th-generation if the price difference is small
