How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
To build an online forum using a LAMP stack (Linux, Apache, MySQL/MariaDB, PHP), you can start with a simple architecture and then add features over time.

Core components

- Linux: Operating system
- Apache: Web server
- MySQL/MariaDB: Database
- PHP: Server-side application logic

Basic forum features

1. User accounts
   - Registration
   - Login/logout
   - Password hashing
   - User profiles

2. Categories
   - General Discussion
   - Support
   - Announcements

3. Threads
   - Users create topics within categories

4. Posts
   - Users reply to threads

5. Moderation
   - Edit/delete posts
   - Lock threads
   - User roles

Database schema example

Users table:

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Categories table:

```sql
CREATE TABLE categories (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT
);
```

Threads table:

```sql
CREATE TABLE threads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    category_id INT NOT NULL,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

Posts table:

```sql
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    thread_id INT NOT NULL,
    user_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (thread_id) REFERENCES threads(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

Database connection example

```php
<?php

$pdo = new PDO(
    "mysql:host=localhost;dbname=forum;charset=utf8mb4",
    "forum_user",
    "password",
    [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
    ]
);
```

User registration example

```php
$passwordHash = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, email, password_hash)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['username'],
    $_POST['email'],
    $passwordHash
]);
```

User login example

```php
$stmt = $pdo->prepare(
    "SELECT * FROM users WHERE username = ?"
);

$stmt->execute([$_POST['username']]);

$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    $_SESSION['user_id'] = $user['id'];
}
```

Creating a thread

```php
$stmt = $pdo->prepare(

     VALUES (?, ?, ?)"

$stmt->execute([
    $_POST['category_id'],
```

Security considerations

- Use prepared statements (PDO or MySQLi)
- Hash passwords with `password_hash()`
- Verify with `password_verify()`
- Escape output with `htmlspecialchars()`
- Use CSRF protection for forms
- Validate all user input
- Limit login attempts
- Use HTTPS

Example displaying a post safely

```php
echo htmlspecialchars($post['content'], ENT_QUOTES, 'UTF-8');
```
