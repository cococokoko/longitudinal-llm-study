Give a numerical example to illustrate the concept of partial derivative.

── Response ────────────────────────────────────────────────────────
A partial derivative measures how a function changes with respect to one variable while holding the other variables constant.

Example:

Consider the function

f(x, y) = x²y + 3y

Suppose we want the partial derivatives at the point (x, y) = (2, 4).

1. Partial derivative with respect to x

Treat y as a constant:

∂f/∂x = 2xy

At (2, 4):

∂f/∂x(2,4) = 2 × 2 × 4 = 16

Interpretation:
Near the point (2,4), if y remains fixed at 4, increasing x by about 1 unit increases f by about 16 units.

For example:

f(2,4) = 2²·4 + 3·4 = 28
f(2.1,4) = 2.1²·4 + 12 = 29.64

Actual change = 29.64 − 28 = 1.64

The derivative predicts:
16 × 0.1 = 1.6

which is close to the actual change.

2. Partial derivative with respect to y

Treat x as a constant:

∂f/∂y = x² + 3

At (2,4):

∂f/∂y(2,4) = 2² + 3 = 7

Interpretation:
Near the point (2,4), if x remains fixed at 2, increasing y by about 1 unit increases f by about 7 units.

For example:

f(2,4) = 28
f(2,4.1) = 2²·4.1 + 3·4.1 = 28.7

Actual change = 28.7 − 28 = 0.7

The derivative predicts:
7 × 0.1 = 0.7

exactly in this case.

So at the point (2,4):
- ∂f/∂x = 16 tells you the sensitivity of f to changes in x while holding y fixed.
- ∂f/∂y = 7 tells you the sensitivity of f to changes in y while holding x fixed.
