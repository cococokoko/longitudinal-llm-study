### Phase 1: The Architecture (Database Schema)
A forum relies on relationships between three core entities: **Users**, **Topics** (threads), and **Posts** (replies). 

First, you will set up your **MySQL** database.

```sql
CREATE DATABASE forum_db;
USE forum_db;

-- 1. Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. Topics Table (The threads)
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);

-- 3. Posts Table (The replies within a thread)
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    topic_id INT NOT NULL,
    user_id INT,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);
```

### Phase 2: The Backend Logic (PHP)
To ensure security, we must prevent **SQL Injection** (using PDO prepared statements) and **XSS (Cross-Site Scripting)** (using `htmlspecialchars`).

#### 1. Database Connection (`db.php`)
```php
<?php
$host = 'localhost';
$db   = 'forum_db';
$user = 'root'; // Replace with your DB username
$pass = '';     // Replace with your DB password
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->code);
}
?>
```

#### 2. User Registration (`register.php`)
*Security Note: Never store plain-text passwords. Use PHP's `password_hash()`.*
```php
<?php
require 'db.php';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $username = trim($_POST['username']);
    $email = trim($_POST['email']);
    $password = password_hash($_POST['password'], PASSWORD_BCRYPT);

    $stmt = $pdo->prepare("INSERT INTO users (username, email, password_hash) VALUES (?, ?, ?)");
    try {
        $stmt->execute([$username, $email, $password]);
        echo "Registration successful! <a href='login.php'>Login here</a>";
    } catch (PDOException $e) {
        echo "Error: Username or Email already exists.";
    }
}
?>

<form method="POST">
    <input type="text" name="username" placeholder="Username" required><br>
    <input type="email" name="email" placeholder="Email" required><br>
    <input type="password" name="password" placeholder="Password" required><br>
    <button type="submit">Register</button>
</form>
```

#### 3. The Forum Homepage (`index.php`)
This page displays all active discussion topics and allows logged-in users to create new ones.
```php
<?php
require 'db.php';
session_start();

// Handle new topic creation
if ($_SERVER['REQUEST_METHOD'] === 'POST' && isset($_SESSION['user_id'])) {
    $title = trim($_POST['title']);
    if (!empty($title)) {
        $stmt = $pdo->prepare("INSERT INTO topics (title, user_id) VALUES (?, ?)");
        $stmt->execute([$title, $_SESSION['user_id']]);
    }
}

// Fetch all topics
$topics = $pdo->query("SELECT topics.*, users.username FROM topics LEFT JOIN users ON topics.user_id = users.id ORDER BY topics.created_at DESC")->fetchAll();
?>

<h1>Law Firm Discussion Forum</h1>

<?php if (isset($_SESSION['username'])): ?>
    <p>Welcome, <?php echo htmlspecialchars($_SESSION['username']); ?>! <a href="logout.php">Logout</a></p>
    <form method="POST">
        <input type="text" name="title" placeholder="Start a new topic..." required>
        <button type="submit">Create Topic</button>
    </form>
<?php else: ?>
    <p><a href="login.php">Login</a> or <a href="register.php">Register</a> to post.</p>
<?php endif; ?>

<h3>Topics</h3>
<ul>
    <?php foreach ($topics as $topic): ?>
        <li>
            <a href="topic.php?id=<?php echo $topic['id']; ?>">
                <?php echo htmlspecialchars($topic['title']); ?>
            </a> 
            (by <?php echo htmlspecialchars($topic['username'] ?? 'Deleted User'); ?>)
        </li>
    <?php endforeach; ?>
</ul>
```

#### 4. Viewing a Topic and Replying (`topic.php`)
```php
<?php
require 'db.php';
session_start();

$topic_id = $_GET['id'] ?? null;
if (!$topic_id) { die("Topic not found."); }

// Handle new reply
if ($_SERVER['REQUEST_METHOD'] === 'POST' && isset($_SESSION['user_id'])) {
    $content = trim($_POST['content']);
    if (!empty($content)) {
        $stmt = $pdo->prepare("INSERT INTO posts (topic_id, user_id, content) VALUES (?, ?, ?)");
        $stmt->execute([$topic_id, $_SESSION['user_id'], $content]);
    }
}

// Fetch topic details
$stmt = $pdo->prepare("SELECT topics.*, users.username FROM topics LEFT JOIN users ON topics.user_id = users.id WHERE topics.id = ?");
$stmt->execute([$topic_id]);
$topic = $stmt->fetch();

// Fetch all posts (replies)
$stmt = $pdo->prepare("SELECT posts.*, users.username FROM posts LEFT JOIN users ON posts.user_id = users.id WHERE posts.topic_id = ? ORDER BY posts.created_at ASC");
$stmt->execute([$topic_id]);
$posts = $stmt->fetchAll();
?>

<h2>Topic: <?php echo htmlspecialchars($topic['title']); ?></h2>
<p><a href="index.php">Back to Home</a></p>

<div class="posts">
    <?php foreach ($posts as $post): ?>
        <div style="border: 1px solid #ccc; padding: 10px; margin-bottom: 10px;">
            <p><strong><?php echo htmlspecialchars($post['username'] ?? 'Deleted User'); ?></strong> wrote:</p>
            <p><?php echo nl2br(htmlspecialchars($post['content'])); ?></p>
            <small><?php echo $post['created_at']; ?></small>
        </div>
    <?php endforeach; ?>
</div>

<?php if (isset($_SESSION['user_id'])): ?>
    <form method="POST">
        <textarea name="content" placeholder="Write a reply..." required></textarea><br>
        <button type="submit">Post Reply</button>
    </form>
<?php endif; ?>
```
