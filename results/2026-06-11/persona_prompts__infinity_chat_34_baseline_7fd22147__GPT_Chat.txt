What is a good secondhand market laptop for learning Python?

── Response ────────────────────────────────────────────────────────
A good secondhand laptop for learning Python doesn't need to be very powerful. For most beginners, the priorities are reliability, battery life, a comfortable keyboard, and enough RAM to run a code editor, browser, and perhaps some development tools at the same time.

Good secondhand options:

- Lenovo ThinkPad T480
  - Often considered one of the best value used laptops.
  - Durable, excellent keyboard, easy to upgrade.
  - Look for:
    - Intel Core i5-8250U or better
    - 16 GB RAM if possible (8 GB minimum)
    - 256 GB SSD or larger

- Lenovo ThinkPad X280 or X1 Carbon (6th Gen or newer)
  - More portable than the T480.
  - Great for coding on the go.
  - X1 Carbon models tend to be more expensive.

- Dell Latitude 5490, 5400, 5410, or 7420
  - Business-class laptops built for long-term use.
  - Usually available at good prices from corporate refurbishers.

- HP EliteBook 840 G5/G6/G7
  - Solid build quality and good keyboards.
  - Often competitively priced in the used market.

- Apple MacBook Air (M1, 2020)
  - If your budget allows, this is one of the best programming laptops available.
  - Excellent battery life and performance.
  - Often found used at reasonable prices compared to newer Macs.
  - Great for Python, web development, and data science.

Minimum specs to aim for:
- CPU: Intel 8th-generation Core i5 or newer, AMD Ryzen 5 or newer, or Apple M1
- RAM: 8 GB minimum, 16 GB preferred
- Storage: 256 GB SSD minimum
- Screen: 1080p (Full HD) if possible

Typical price ranges (depending on location):
- $150–250: ThinkPad T480, Latitude 5490, EliteBook 840 G5
- $250–400: Newer Latitude/EliteBook models, ThinkPad T14 Gen 1
- $450–700: Used MacBook Air M1

If you're learning Python specifically (automation, scripting, web development, beginner data analysis), a refurbished ThinkPad T480 with 16 GB RAM and a 256–512 GB SSD is often the best value. It can comfortably run Windows, Linux, VS Code, Jupyter notebooks, and most beginner-to-intermediate Python projects.
