# Making a Zombie Survivor Game on Scratch

Here's a straightforward guide to build a functional zombie survivor game!

---

## Basic Game Concept
- Player moves and shoots zombies
- Zombies chase the player
- Score increases per kill
- Game ends when zombies catch the player

---

## Step 1: Set Up Your Project

1. Go to **scratch.mit.edu**
2. Create a new project
3. Plan your sprites:
   - Player character
   - Zombie
   - Bullet
   - (Optional) Health bar, score display

---

## Step 2: Player Sprite

**Movement script:**
```
When [Green Flag] clicked
Forever
  If [W key pressed] → change y by 3
  If [S key pressed] → change y by -3
  If [A key pressed] → change x by -3
  If [D key pressed] → change x by 3
```

**Shooting (point toward mouse):**
```
When [Green Flag] clicked
Forever
  Point toward [mouse pointer]
  If [space pressed] → broadcast [shoot]
```

---

## Step 3: Bullet Sprite

```
When I receive [shoot]
  Go to [Player]
  Point toward [mouse pointer]
  Show
  Repeat until touching edge
    Move 10 steps
    If touching [Zombie] → 
      broadcast [zombie hit]
      Hide
```

---

## Step 4: Zombie Sprite

```
When [Green Flag] clicked
Forever
  Point toward [Player]
  Move 1 step
  If touching [Player] →
    broadcast [game over]
```

**Use clones for multiple zombies:**
```
When [Green Flag] clicked
Forever
  Create clone of [Zombie]
  Wait 2 seconds
```

**Each clone:**
```
When I start as a clone
  Go to random position
  Show
  Forever
    Point toward Player
    Move 1 step
```

---

## Step 5: Score & Game Over

**Score variable:**
```
When I receive [zombie hit]
  Change [score] by 1
  Delete this clone
```

**Game Over:**
```
When I receive [game over]
  Stop all
  Show "GAME OVER" costume
```
