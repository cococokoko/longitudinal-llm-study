- A → Space
- B (trigger) → Left Ctrl

- D-Pad → Arrow keys

  - Cursor Up → Mouse Y-
  - Cursor Down → Mouse Y+

- Map Shake X/Y/Z to convenient keys (for example, Q, W, E).
- Map Tilt directions to additional keys if needed.

Using the Windows on-screen keyboard:

1. Press Win + Ctrl + O to open the on-screen keyboard.
2. Keep it docked to one side of the screen.
3. Tap keys on the on-screen keyboard while using the touchpad to aim or point.

Surface-specific tips:

- Enable "Tap to click" in Windows touchpad settings for easier pointer interaction.
- If a game relies heavily on motion controls (Wii Sports, Skyward Sword, etc.), a real controller or Bluetooth Wii Remote is usually much easier.
