── Prompt ──────────────────────────────────────────────────────────
Give a numerical example to illustrate the concept of partial derivative.

── Response ────────────────────────────────────────────────────────
A partial derivative measures how a function changes with respect to one variable while keeping the other variables constant.

Example:

Consider the function

f(x, y) = x²y + 3xy

Find the partial derivatives with respect to x and y.

1. Partial derivative with respect to x

Treat y as a constant:

∂f/∂x = ∂/∂x (x²y + 3xy)
       = 2xy + 3y

Now evaluate at the point (x, y) = (2, 4):

∂f/∂x(2,4) = 2(2)(4) + 3(4)
            = 16 + 12
            = 28

Interpretation:
When y is fixed at 4, a small increase in x near x = 2 increases the function by approximately 28 times the change in x.

2. Partial derivative with respect to y

Treat x as a constant:

∂f/∂y = ∂/∂y (x²y + 3xy)
       = x² + 3x

Now evaluate at (2,4):

∂f/∂y(2,4) = 2² + 3(2)
            = 4 + 6
            = 10

Interpretation:
When x is fixed at 2, a small increase in y near y = 4 increases the function by approximately 10 times the change in y.

Summary at (2,4):
- ∂f/∂x = 28
- ∂f/∂y = 10

This shows that the function is more sensitive to changes in x than to changes in y at the point (2,4).
