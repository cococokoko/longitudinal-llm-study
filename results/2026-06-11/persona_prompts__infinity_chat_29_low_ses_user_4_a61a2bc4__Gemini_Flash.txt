Building an online forum using the **LAMP stack** (Linux, Apache, MySQL, PHP) is a classic, highly rewarding web development project. It will teach you the core fundamentals of how the internet works: databases, server-side logic, and user interaction.

Here is a step-by-step guide to building a basic, functional forum from scratch.

---

### Phase 1: Setting Up Your Environment (The "L" and "A")
Before writing code, you need a local server environment on your computer to run PHP and MySQL.

*   **If you use Windows:** Download and install **XAMPP** or **WampServer**.
*   **If you use Mac:** Download and install **MAMP**.
*   **If you use Linux:** You can install Apache, MySQL, and PHP directly via your terminal (`sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql`).

Once installed, start the **Apache** and **MySQL** services through the control panel of the software you downloaded.

---

### Phase 2: Designing the Database (The "M")
A forum needs to store users, categories, topics (threads), and posts (replies). 

Open your browser and go to `http://localhost/phpmyadmin`. Create a new database named `forum_db`, and run the following SQL queries to create your tables:

```sql
-- 1. Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. Topics Table (The threads)
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    subject VARCHAR(255) NOT NULL,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

-- 3. Posts Table (The replies inside threads)
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    content TEXT NOT NULL,
    topic_id INT,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

---

### Phase 3: Writing the PHP Code (The "P")
Go to your local server's root directory (usually `C:\xampp\htdocs` on Windows or `/Applications/MAMP/htdocs` on Mac). Create a new folder named `forum`. Inside this folder, we will create 5 essential files.

#### 1. `db.php` (Database Connection)
This file connects your PHP code to your MySQL database.
```php
<?php
$host = 'localhost';
$db_user = 'root';
$db_pass = ''; // Default is empty for XAMPP/MAMP
$db_name = 'forum_db';

$conn = new mysqli($host, $db_user, $db_pass, $db_name);

if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
session_start(); // Starts user sessions
?>
```

#### 2. `register.php` (User Registration)
This allows users to create an account. We use `password_hash` to keep passwords secure.
```php
<?php
include 'db.php';

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = password_hash($_POST['password'], PASSWORD_BCRYPT);

    $stmt = $conn->prepare("INSERT INTO users (username, password) VALUES (?, ?)");
    $stmt->bind_param("ss", $username, $password);
    
    if ($stmt->execute()) {
        echo "Registration successful! <a href='login.php'>Login here</a>";
    } else {
        echo "Error: Username might already exist.";
    }
}
?>
```

#### 3. `login.php` (User Login)
This verifies the user and logs them in.
```php
<?php
include 'db.php';

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = $_POST['password'];

    $stmt = $conn->prepare("SELECT id, password FROM users WHERE username = ?");
    $stmt->bind_param("s", $username);
    $stmt->execute();
    $result = $stmt->get_result();
    
    if ($user = $result->fetch_assoc()) {
        if (password_verify($password, $user['password'])) {
            $_SESSION['user_id'] = $user['id'];
            $_SESSION['username'] = $username;
            header("Location: index.php");
            exit;
        }
    }
    echo "Invalid username or password.";
}
?>
<form method="POST">
    <h2>Login</h2>
    <input type="text" name="username" placeholder="Username" required><br><br>
    <input type="password" name="password" placeholder="Password" required><br><br>
    <button type="submit">Login</button>
</form>
```
