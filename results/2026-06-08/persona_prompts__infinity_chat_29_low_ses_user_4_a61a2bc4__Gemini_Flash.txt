── Response ────────────────────────────────────────────────────────
Working shifts at a hospital means your free time comes in unpredictable blocks. Coding a forum from scratch is a fantastic way to learn the **LAMP** stack (**L**inux, **A**pache, **M**ySQL, **P**HP), but it can be overwhelming. 

To make this manageable for your schedule, here is a step-by-step guide to building a **simplified, working forum** from scratch. 

---

### Phase 1: Set Up Your Environment (The "L" and "A")
Before writing code, you need a local server environment. 
*   **The Easy Way:** Download and install **XAMPP** (Windows/Linux) or **MAMP** (Mac). This installs Apache, MySQL, and PHP on your computer with one click.
*   Once installed, open the control panel and start **Apache** and **MySQL**.
*   Your website files will live in the `htdocs` folder (inside the XAMPP directory). Create a new folder there named `forum`.

---

### Phase 2: Design the Database (The "M")
A forum needs to store data. We will create two tables: `topics` (the threads) and `replies`.

1. Open your browser and go to `http://localhost/phpmyadmin`.
2. Create a new database named `simple_forum`.
3. Click the "SQL" tab and run this code to create your tables:

```sql
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE replies (
    id INT AUTO_INCREMENT PRIMARY KEY,
    topic_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE
);
```

---

### Phase 3: Write the Code (The "P")
Inside your `htdocs/forum` folder, you will create three files.

#### File 1: `db.php` (The Database Connection)
This file connects PHP to your MySQL database. We use **PDO** because it is secure.

```php
<?php
$host = 'localhost';
$db   = 'simple_forum';
$user = 'root'; // Default XAMPP user
$pass = '';     // Default XAMPP password is empty
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
?>
```
