How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
Building a forum from scratch using the **LAMP stack** (Linux, Apache, MySQL, PHP) is an excellent way to learn web development. 

Below is a step-by-step guide to building a **Minimum Viable Product (MVP)** for a forum. This includes the database design, database connection, and the PHP code to display and create threads.

---

### Phase 1: Database Design (MySQL)
A forum relies heavily on relational data. We need three basic tables: **Users**, **Topics** (threads), and **Posts** (replies).

Run this SQL in your MySQL database (e.g., via phpMyAdmin or MySQL CLI):

```sql
CREATE DATABASE lamp_forum;
USE lamp_forum;

-- 1. Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. Topics Table (The threads)
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    subject VARCHAR(255) NOT NULL,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);

-- 3. Posts Table (The replies inside a thread)
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    content TEXT NOT NULL,
    topic_id INT,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);
```

---

### Phase 2: Database Connection (`db.php`)
We will use **PDO (PHP Data Objects)** because it is secure and supports prepared statements (which prevent SQL injection).

Create a file named `db.php`:

```php
<?php
$host = 'localhost';
$db   = 'lamp_forum';
$user = 'root'; // Change to your MySQL username
$pass = '';     // Change to your MySQL password
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
?>
```
