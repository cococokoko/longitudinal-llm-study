── Prompt ──────────────────────────────────────────────────────────
How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
Building an online forum from scratch using the **LAMP** stack (Linux, Apache, MySQL, PHP) is a fantastic way to learn full-stack web development. 

Here is a step-by-step guide to building a basic, functional forum.

---

### Step 1: Set Up Your LAMP Environment
If you are on a local machine, the easiest way to get started is by installing a pre-packaged LAMP environment:
*   **Windows:** Install [XAMPP](https://www.apachefriends.org/index.html) or Laragon.
*   **Mac:** Install MAMP or use Homebrew.
*   **Linux:** Install Apache, MySQL, and PHP via your package manager (e.g., `sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql`).

---

### Step 2: Design the Database (MySQL)
A basic forum needs three main tables: **Users**, **Topics** (Threads), and **Posts** (Replies). 

Open your MySQL tool (like phpMyAdmin or the command line) and run this SQL:

```sql
CREATE DATABASE simple_forum;
USE simple_forum;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL, -- Will store hashed passwords
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    user_id INT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    content TEXT NOT NULL,
    topic_id INT NOT NULL,
    user_id INT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

---

### Step 3: Project Structure
Create a folder in your web root (e.g., `htdocs/forum` or `/var/www/html/forum`).
Organize your files like this:
*   `db.php` (Database connection)
*   `index.php` (Home page, lists topics)
*   `register.php` (User sign up)
*   `login.php` (User login)
*   `topic.php` (Views a specific topic and its posts)
*   `create_topic.php` (Form to make a new topic)

---

### Step 4: Write the PHP Code

#### 1. Database Connection (`db.php`)
Always use **PDO** (PHP Data Objects) for database connections. It is secure and supports prepared statements.

```php
<?php
$host = '127.0.0.1';
$db   = 'simple_forum';
$user = 'root'; // Change to your DB username
$pass = '';     // Change to your DB password
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
];

try {
    $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
    throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
session_start(); // Start session for user login tracking
?>
```

#### 2. User Registration (`register.php`)
Never store plain-text passwords. Always use PHP's `password_hash()`.

```php
<?php
require 'db.php';

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = password_hash($_POST['password'], PASSWORD_DEFAULT);

    $stmt = $pdo->prepare("INSERT INTO users (username, password) VALUES (?, ?)");
    try {
        $stmt->execute([$username, $password]);
        echo "Registration successful! <a href='login.php'>Login here</a>";
    } catch (PDOException $e) {
        echo "Username already exists.";
    }
}
?>
<form method="POST">
    Username: <input type="text" name="username" required><br>
    Password: <input type="password" name="password" required><br>
    <button type="submit">Register</button>
</form>
```
