Minimum specifications I'd look for:
- CPU: Intel Core i5 8th generation or newer (or AMD Ryzen 5)
- RAM: 8 GB minimum; 16 GB preferred
- Storage: 256 GB SSD minimum
- Display: 1920×1080 (Full HD) if possible
- Battery: Check health if buying locally

- Python itself runs well on almost any laptop from the last 8–10 years.
- If you later want to do data science, machine learning, Docker, or run virtual machines, 16 GB RAM becomes much more important.
