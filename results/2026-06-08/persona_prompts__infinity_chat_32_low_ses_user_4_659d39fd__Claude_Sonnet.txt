# Making a Zombie Survivor Game on Scratch

Here's a straightforward guide to building a basic zombie survival game!

---

## Basic Game Concept
- Player moves around avoiding/fighting zombies
- Zombies chase the player
- Player has health points
- Survive as long as possible

---

## Step 1: Set Up Your Sprites

**You'll need:**
- Player sprite
- Zombie sprite
- Bullet/weapon sprite (optional)
- Health bar

---

## Step 2: Player Movement

Add this to your **Player sprite:**

```
when green flag clicked
forever
  if <key [w] pressed> then
    change y by 3
  end
  if <key [s] pressed> then
    change y by -3
  end
  if <key [a] pressed> then
    change x by -3
  end
  if <key [d] pressed> then
    change x by 3
  end
end
```

---

## Step 3: Zombie AI (Chasing Player)

Add this to your **Zombie sprite:**

```
when green flag clicked
forever
  point towards [Player]
  move 2 steps
end
```

---

## Step 4: Health System

**Create a variable called `Health`**

```
when green flag clicked
set [Health] to 100

// In zombie sprite:
if <touching [Player]> then
  change [Health] by -1
  wait 0.5 seconds
end

// Game over check:
if <(Health) < 1> then
  broadcast [Game Over]
  stop all
end
```

---

## Step 5: Shooting Mechanic

On **mouse click**, create a clone of a bullet sprite:

```
when green flag clicked
forever
  if <mouse down?> then
    create clone of [Bullet]
    wait 0.3 seconds
  end
end

// In Bullet sprite:
when I start as a clone
point towards [mouse pointer]
repeat until <touching edge?>
  move 10 steps
  if <touching [Zombie]> then
    // delete zombie clone
    delete this clone
  end
end
delete this clone
```

---

## Step 6: Multiple Zombies

Use **clones** to spawn many zombies:

```
when green flag clicked
forever
  create clone of [Zombie]
  wait 2 seconds
end

when I start as a clone
go to [random position]
forever
  point towards [Player]
  move 2 steps
end
```

---

## Step 7: Score System

```
when green flag clicked
set [Score] to 0

// When zombie is defeated:
change [Score] by 10
```
