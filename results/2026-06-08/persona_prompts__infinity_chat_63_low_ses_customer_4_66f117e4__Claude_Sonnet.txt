# Partial Derivatives: A Numerical Example

## Setup

Suppose a hospital patient care aide's **weekly earnings** depend on two variables:

- **h** = regular hours worked
- **n** = number of night shift hours worked (which pay a bonus)

The earnings function is:

$$E(h, n) = 15h + 22n + 0.5n^2$$

where $15 is the regular hourly rate, $22 is the night shift rate, and the $0.5n²$ term reflects overtime bonuses.

---

## Computing the Partial Derivatives

### ∂E/∂h (rate of change with respect to regular hours, holding n constant)

$$\frac{\partial E}{\partial h} = 15$$

### ∂E/∂n (rate of change with respect to night hours, holding h constant)

$$\frac{\partial E}{\partial n} = 22 + n$$

---

## Numerical Illustration

Suppose in a given week: **h = 32**, **n = 8**

**Current earnings:**
$$E(32, 8) = 15(32) + 22(8) + 0.5(8)^2 = 480 + 176 + 32 = \$688$$

### What if you work **one more regular hour** (h = 33, n = 8)?

$$E(33, 8) = 15(33) + 22(8) + 0.5(64) = 495 + 176 + 32 = \$707$$

> Increase = **$19... wait:** $707 - $688 = **$19? No → $15** ✓

$$\frac{\partial E}{\partial h} = 15 \quad \checkmark$$

### What if you work **one more night hour** (h = 32, n = 9)?

$$E(32, 9) = 15(32) + 22(9) + 0.5(81) = 480 + 198 + 40.5 = \$718.50$$

> Increase = $718.50 - $688 = **$30.50**

Using the formula: $\frac{\partial E}{\partial n} = 22 + n = 22 + 8 = \mathbf{30}$ *(close, exact for infinitesimal change)*

---

## Key Takeaway

| Partial Derivative | Meaning | Value at (32, 8) |
|---|---|---|
| ∂E/∂h | Each extra regular hour earns... | **$15** |
| ∂E/∂n | Each extra night hour earns... | **$30** |

> **The partial derivative isolates the effect of changing ONE variable while keeping all others fixed** — exactly like asking *"what does one more night shift hour do to my paycheck?"*
