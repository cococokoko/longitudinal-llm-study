### Phase 1: Database Design (MySQL)
A basic forum needs three core tables: **Users**, **Categories/Topics**, and **Posts**.

Run this SQL in your MySQL database (e.g., via phpMyAdmin or the command line):

```sql
CREATE DATABASE forum_db;
USE forum_db;

-- 1. Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. Topics Table (The main threads)
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);

-- 3. Posts Table (The replies inside a thread)
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    topic_id INT NOT NULL,
    user_id INT,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL
);
```

### Phase 2: Database Connection (PHP)
Create a file named `db.php`. This will be included at the top of your other files to connect to MySQL.

```php
<?php
// db.php
$host = 'localhost';
$db   = 'forum_db';
$user = 'root'; // Change to your MySQL username
$pass = '';     // Change to your MySQL password
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
?>
```

### Phase 3: User Authentication (Registration & Login)
To post on a forum, users need to log in. We will use PHP sessions.

#### `register.php`
```php
<?php
require 'db.php';
if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = password_hash($_POST['password'], PASSWORD_BCRYPT); // Secure password hashing

    $stmt = $pdo->prepare("INSERT INTO users (username, password) VALUES (?, ?)");
    try {
        $stmt->execute([$username, $password]);
        header("Location: login.php");
        exit;
    } catch (PDOException $e) {
        $error = "Username already exists.";
    }
}
?>
<form method="POST">
    <h2>Register</h2>
    <?php if(isset($error)) echo "<p>$error</p>"; ?>
    <input type="text" name="username" placeholder="Username" required><br><br>
    <input type="password" name="password" placeholder="Password" required><br><br>
    <button type="submit">Register</button>
</form>
<p>Already have an account? <a href="login.php">Login here</a>.</p>
```

#### `login.php`
```php
<?php
require 'db.php';
session_start();

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = $_POST['password'];

    $stmt = $pdo->prepare("SELECT * FROM users WHERE username = ?");
    $stmt->execute([$username]);
    $user = $stmt->fetch();

    if ($user && password_verify($password, $user['password'])) {
        $_SESSION['user_id'] = $user['id'];
        $_SESSION['username'] = $user['username'];
        header("Location: index.php");
        exit;
    } else {
        $error = "Invalid username or password.";
    }
}
?>
<form method="POST">
    <h2>Login</h2>
    <?php if(isset($error)) echo "<p>$error</p>"; ?>
    <input type="text" name="username" placeholder="Username" required><br><br>
    <input type="password" name="password" placeholder="Password" required><br><br>
    <button type="submit">Login</button>
</form>
<p>Don't have an account? <a href="register.php">Register here</a>.</p>
```

### Phase 4: The Forum Homepage (`index.php`)
This page displays all existing topics and allows logged-in users to create a new topic.

```php
<?php
require 'db.php';
session_start();

// Handle creating a new topic
if ($_SERVER['REQUEST_METHOD'] == 'POST' && isset($_SESSION['user_id'])) {
    $title = $_POST['title'];
    $content = $_POST['content'];
    $user_id = $_SESSION['user_id'];

    // 1. Insert Topic
    $stmt = $pdo->prepare("INSERT INTO topics (title, user_id) VALUES (?, ?)");
    $stmt->execute([$title, $user_id]);
    $topic_id = $pdo->lastInsertId();

    // 2. Insert the first post of the topic
    $stmt = $pdo->prepare("INSERT INTO posts (topic_id, user_id, content) VALUES (?, ?, ?)");
    $stmt->execute([$topic_id, $user_id, $content]);
}

// Fetch all topics
$stmt = $pdo->query("SELECT topics.*, users.username FROM topics JOIN users ON topics.user_id = users.id ORDER BY topics.created_at DESC");
$topics = $stmt->fetchAll();
?>

<!DOCTYPE html>
<html>
<head><title>My LAMP Forum</title></head>
<body>
    <header>
        <h1>Welcome to the Forum</h1>
        <?php if (isset($_SESSION['username'])): ?>
            <p>Logged in as: <strong><?php echo htmlspecialchars($_SESSION['username']); ?></strong> | <a href="logout.php">Logout</a></p>
        <?php else: ?>
            <p><a href="login.php">Login</a> or <a href="register.php">Register</a> to post.</p>
        <?php endif; ?>
    </header>

    <hr>

    <!-- Create Topic Form -->
    <?php if (isset($_SESSION['user_id'])): ?>
        <h3>Create a New Topic</h3>
        <form method="POST">
            <input type="text" name="title" placeholder="Topic Title" required style="width: 300px;"><br><br>
            <textarea name="content" placeholder="Write your first post here..." required style="width: 300px; height: 100px;"></textarea><br><br>
            <button type="submit">Create Topic</button>
        </form>
        <hr>
    <?php endif; ?>

    <!-- Topics List -->
    <h3>Discussion Threads</h3>
    <ul>
        <?php foreach ($topics as $topic): ?>
            <li>
                <a href="view_topic.php?id=<?php echo $topic['id']; ?>">
                    <strong><?php echo htmlspecialchars($topic['title']); ?></strong>
                </a> 
                by <?php echo htmlspecialchars($topic['username']); ?> 
                on <?php echo $topic['created_at']; ?>
            </li>
        <?php endforeach; ?>
    </ul>
</body>
</html>
```

This page displays a specific topic, all its replies, and a form to add a new reply.

```php
<?php
require 'db.php';
session_start();

$topic_id = $_GET['id'] ?? null;
if (!$topic_id) {
