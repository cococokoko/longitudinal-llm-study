Below is a structured, academic guide to architecting and coding a minimalist, functional forum from scratch.

### Phase 1: Database Architecture (MySQL)
A relational database is the backbone of a forum. At a minimum, we need three entities: **Users**, **Topics** (threads), and **Posts** (replies).

Connect to your MySQL instance and run the following schema definition:

```sql
CREATE DATABASE IF NOT EXISTS forum_db;
USE forum_db;

-- 1. Users Table
CREATE TABLE users (
    user_id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB;

-- 2. Topics Table (The threads)
CREATE TABLE topics (
    topic_id INT AUTO_INCREMENT PRIMARY KEY,
    subject VARCHAR(255) NOT NULL,
    created_by INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (created_by) REFERENCES users(user_id) ON DELETE SET NULL
) ENGINE=InnoDB;

-- 3. Posts Table (The replies within threads)
CREATE TABLE posts (
    post_id INT AUTO_INCREMENT PRIMARY KEY,
    post_content TEXT NOT NULL,
    topic_id INT NOT NULL,
    created_by INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(topic_id) ON DELETE CASCADE,
    FOREIGN KEY (created_by) REFERENCES users(user_id) ON DELETE SET NULL
) ENGINE=InnoDB;
```

### Phase 2: Database Connection (PHP)
Create a centralized configuration file (`config.php`) to handle the database connection using **PDO** (PHP Data Objects), which is safer and more modern than the legacy `mysqli` extension.

```php
<?php
// config.php
$host = 'localhost';
$db   = 'forum_db';
$user = 'your_db_user';
$pass = 'your_db_password';
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->getCode());
}

// Start session globally
if (session_status() == PHP_SESSION_NONE) {
    session_start();
}
?>
```

### Phase 3: User Authentication
To post, users must register and log in. We must use secure password hashing (`password_hash` and `password_verify`).

#### Register (`register.php`)
```php
<?php
require 'config.php';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $username = trim($_POST['username']);
    $email = trim($_POST['email']);
    $password = $_POST['password'];

    if (!empty($username) && !empty($email) && !empty($password)) {
        $hash = password_hash($password, PASSWORD_BCRYPT);
        
        $stmt = $pdo->prepare("INSERT INTO users (username, email, password_hash) VALUES (?, ?, ?)");
        try {
            $stmt->execute([$username, $email, $hash]);
            header("Location: login.php?success=registered");
            exit;
        } catch (PDOException $e) {
            $error = "Username or Email already exists.";
        }
    } else {
        $error = "All fields are required.";
    }
}
?>
<!-- HTML Form -->
<form method="POST">
    <h2>Register</h2>
    <?php if(isset($error)) echo "<p style='color:red;'>$error</p>"; ?>
    <input type="text" name="username" placeholder="Username" required><br>
    <input type="email" name="email" placeholder="Email" required><br>
    <input type="password" name="password" placeholder="Password" required><br>
    <button type="submit">Register</button>
</form>
```

#### Login (`login.php`)
```php
<?php
require 'config.php';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $username = trim($_POST['username']);
    $password = $_POST['password'];

    $stmt = $pdo->prepare("SELECT * FROM users WHERE username = ?");
    $stmt->execute([$username]);
    $user = $stmt->fetch();

    if ($user && password_verify($password, $user['password_hash'])) {
        $_SESSION['user_id'] = $user['user_id'];
        $_SESSION['username'] = $user['username'];
        header("Location: index.php");
        exit;
    } else {
        $error = "Invalid credentials.";
    }
}
?>
<!-- HTML Form -->
<form method="POST">
    <h2>Login</h2>
    <?php if(isset($error)) echo "<p style='color:red;'>$error</p>"; ?>
    <input type="text" name="username" placeholder="Username" required><br>
    <input type="password" name="password" placeholder="Password" required><br>
    <button type="submit">Login</button>
</form>
```

### Phase 4: The Forum Index (`index.php`)
This page displays all active discussion threads and allows logged-in users to create a new topic.

```php
<?php
require 'config.php';

// Handle new topic creation
if ($_SERVER['REQUEST_METHOD'] === 'POST' && isset($_SESSION['user_id'])) {
    $subject = htmlspecialchars(trim($_POST['subject']));
    $first_post = htmlspecialchars(trim($_POST['content']));

    if (!empty($subject) && !empty($first_post)) {
        $pdo->beginTransaction();
        try {
            // Insert Topic
            $stmt = $pdo->prepare("INSERT INTO topics (subject, created_by) VALUES (?, ?)");
            $stmt->execute([$subject, $_SESSION['user_id']]);
            $topic_id = $pdo->lastInsertId();

            // Insert First Post
            $stmt2 = $pdo->prepare("INSERT INTO posts (post_content, topic_id, created_by) VALUES (?, ?, ?)");
            $stmt2->execute([$first_post, $topic_id, $_SESSION['user_id']]);

            $pdo->commit();
            header("Location: topic.php?id=" . $topic_id);
            exit;
        } catch (Exception $e) {
            $pdo->rollBack();
            $error = "Failed to create topic.";
        }
    }
}

// Fetch all topics
$stmt = $pdo->query("SELECT t.*, u.username FROM topics t LEFT JOIN users u ON t.created_by = u.user_id ORDER BY t.created_at DESC");
$topics = $stmt->fetchAll();
?>

<!DOCTYPE html>
<html>
<head><title>LAMP Forum</title></head>
<body>
    <header>
        <h1>Welcome to the Forum</h1>
        <?php if (isset($_SESSION['user_id'])): ?>
            <p>Logged in as: <strong><?php echo $_SESSION['username']; ?></strong> | <a href="logout.php">Logout</a></p>
        <?php else: ?>
            <p><a href="login.php">Login</a> or <a href="register.php">Register</a> to post.</p>
        <?php endif; ?>
    </header>

    <main>
        <h2>Discussion Threads</h2>
        <ul>
            <?php foreach ($topics as $topic): ?>
                <li>
                    <a href="topic.php?id=<?php echo $topic['topic_id']; ?>">
                        <?php echo htmlspecialchars($topic['subject']); ?>
                    </a> 
                    by <?php echo htmlspecialchars($topic['username'] ?? 'Deleted User'); ?> 
                    on <?php echo $topic['created_at']; ?>
                </li>
            <?php endforeach; ?>
        </ul>

        <?php
