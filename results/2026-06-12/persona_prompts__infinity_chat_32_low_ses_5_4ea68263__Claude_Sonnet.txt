# Making a Zombie Survivor Game on Scratch

## Basic Setup

### Sprites You'll Need
- **Player character**
- **Zombie sprite**
- **Bullet/projectile**
- **Background** (top-down view works well)

---

## Step-by-Step Guide

### 1. Player Movement
Add this to your **Player sprite**:

```
when green flag clicked
forever
  if <key [w] pressed> then
    change y by (5)
  end
  if <key [s] pressed> then
    change y by (-5)
  end
  if <key [a] pressed> then
    change x by (-5)
  end
  if <key [d] pressed> then
    change x by (5)
  end
end
```

### 2. Player Shooting
```
when [space] key pressed
create clone of [Bullet]

-- On Bullet sprite:
when I start as a clone
point in direction (player direction)
repeat until <touching edge?>
  move (10) steps
  if <touching [Zombie]> then
    broadcast [hit zombie]
    delete this clone
  end
end
delete this clone
```

### 3. Zombie AI
```
when green flag clicked
forever
  point towards [Player]
  move (2) steps
  if <touching [Player]> then
    broadcast [player hit]
  end
end
```

### 4. Spawning Multiple Zombies
```
when green flag clicked
forever
  wait (2) seconds
  create clone of [Zombie]

when I start as a clone
go to (random position)
-- then run zombie AI
```

### 5. Health System
```
-- Create a variable called "Health"
when green flag clicked
set [Health] to (3)

when I receive [player hit]
change [Health] by (-1)
if <(Health) = (0)> then
  broadcast [game over]
end
```

### 6. Score System
```
-- Create a variable called "Score"
when I receive [hit zombie]
change [Score] by (10)
delete this clone
```
