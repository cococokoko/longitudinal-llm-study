2. Map Wii Remote buttons to keyboard keys
In the Wii Remote configuration window, click each control and press a key on the on-screen keyboard.

A common layout is:
- A → Z
- B (trigger) → X
- 1 → Q
- 2 → W
- Plus (+) → Enter
- Minus (-) → Backspace
- Home → H
- D-Pad → Arrow keys

3. Map Wii Remote motion to touchpad-friendly controls
Many Wii games require pointing and motion.

For IR (pointer):
- Under Point, bind:
  - Up/Down/Left/Right to arrow keys or nearby keys.

For Swing:
- Assign keys such as:
  - Swing Up → I
  - Swing Down → K
  - Swing Left → J
  - Swing Right → L

For Tilt:
- Assign:
  - Tilt Forward → U
  - Tilt Backward → O
  - Tilt Left → N
  - Tilt Right → M

For Shake:
- Assign a convenient key such as Space.

4. Use the Surface touchpad as a mouse pointer
If the game uses Wii pointer controls:
- In the Emulated Wii Remote configuration, bind the pointer axes to mouse movement if available in your Dolphin version.
- Then move the Surface touchpad to aim the Wii cursor.

5. Open the Windows on-screen keyboard
- Press Win + Ctrl + O to toggle the on-screen keyboard.
