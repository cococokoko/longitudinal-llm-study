One example of an optimization technique used in machine learning is Gradient Descent.

Gradient Descent is an algorithm that iteratively adjusts a model's parameters to minimize a loss function by moving in the direction of the negative gradient. Variants include:
- Stochastic Gradient Descent (SGD)
- Mini-batch Gradient Descent
- Adam
- RMSProp

Example: Training a neural network often uses Adam, which is an optimization algorithm based on gradient descent.
