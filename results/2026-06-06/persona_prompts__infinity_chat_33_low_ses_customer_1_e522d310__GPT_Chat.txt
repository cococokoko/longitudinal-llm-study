You can run many Wii games in Dolphin using a Microsoft Surface touchpad and the Windows on-screen keyboard, although some games will be easier than others.

Setup in Dolphin:

1. Open Dolphin.
2. Go to:
   - Controllers
   - Under "Wii Remotes," set "Emulated Wii Remote" for Wii games.
   - Click "Configure."

Suggested Surface touchpad + keyboard mapping:

Pointer (Wii Remote aiming):
- Enable "Cursor" controls in the Emulated Wii Remote settings.
- Map:
  - Cursor Up = Mouse Y-
  - Cursor Down = Mouse Y+
  - Cursor Left = Mouse X-
  - Cursor Right = Mouse X+

This lets the touchpad act as the Wii pointer.

Wii Remote buttons:
- A = Enter
- B (trigger) = Space
- 1 = 1
- 2 = 2
- Plus (+) = +
- Minus (-) = -
- Home = H

D-Pad:
- Up = Arrow Up
- Down = Arrow Down
- Left = Arrow Left
- Right = Arrow Right

Motion controls:
For games that require shaking or tilting:
- Shake X = Q
- Shake Y = W
- Shake Z = E
- Tilt Left = A
- Tilt Right = D
- Tilt Forward = S
- Tilt Backward = X

Using the Windows On-Screen Keyboard:
- Press Win + Ctrl + O to open it.
- Keep it docked on part of the screen.
- Tap keys on the on-screen keyboard when a game needs buttons you don't want to assign to physical keys.

Touchpad clicking:
- A common setup is:
  - Touchpad click = A
  - Keyboard Space = B
- If Dolphin doesn't detect touchpad clicks directly, map A to Enter or another key instead.

For games requiring a Nunchuk:
1. In the Wii Remote configuration window:
   - Extension → Nunchuk.
2. Map:
   - Stick = WASD
   - C = C
   - Z = Z

Games that work reasonably well with this setup:
- New Super Mario Bros. Wii
- Kirby's Return to Dream Land
- Mario Kart Wii (with keyboard steering)
- Many WiiWare titles

Games that are difficult:
- Skyward Sword
- Wii Sports Resort
- Any game that heavily depends on MotionPlus or precise motion controls
