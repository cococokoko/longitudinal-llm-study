── Prompt ──────────────────────────────────────────────────────────
I work shifts at a local hospital as a patient care aide.

How can I make a zombie survivor game on Scratch?

── Response ────────────────────────────────────────────────────────
You can make a simple zombie survivor game in Scratch by building it in small pieces. Since you work shifts, you might find it easiest to create one feature at a time during short sessions.

Basic game idea:
- Player moves around the screen.
- Zombies chase the player.
- Player can shoot zombies.
- Survive as long as possible and earn points.

Step 1: Create the player

Add a player sprite and use keyboard controls.

Player script:

When Green Flag Clicked
- Go to x: 0 y: 0
- Forever
  - If key (W) pressed → change y by 4
  - If key (S) pressed → change y by -4
  - If key (A) pressed → change x by -4
  - If key (D) pressed → change x by 4

You can also use arrow keys instead.

Step 2: Create zombies

Add a zombie sprite.

Zombie script:

When Green Flag Clicked
- Forever
  - Point towards Player
  - Move 2 steps

This makes the zombie constantly chase the player.

Step 3: Spawn zombies

Create variables:
- Score
- Health

Create a "Spawn Zombie" broadcast.

Stage script:

When Green Flag Clicked
- Set Score to 0
- Set Health to 100
- Forever
  - Wait 2 seconds
  - Broadcast Spawn Zombie

Zombie sprite:

When I Receive Spawn Zombie
- Create Clone of Myself

When I Start as a Clone
- Go to random edge
- Show
- Forever
  - Point towards Player
  - Move 2 steps

Step 4: Damage the player

In zombie clone script:

Forever
- If touching Player then
  - Change Health by -1
  - Wait 0.2 seconds

When Health reaches 0:

Stage script:

Forever
- If Health ≤ 0 then
  - Broadcast Game Over
  - Stop All

Step 5: Add shooting

Create a Bullet sprite.

Player script:

Forever
- If mouse down then
  - Create Clone of Bullet
  - Wait 0.2 seconds

Bullet clone:

When I Start as a Clone
- Go to Player
- Point towards mouse-pointer
- Repeat 50
  - Move 10 steps
  - If touching Zombie then
    - Change Score by 1
    - Delete zombie clone
    - Delete this clone
- Delete this clone

Step 6: Increase difficulty

As score rises:
- Spawn zombies faster.
- Increase zombie speed.
- Spawn multiple zombies at once.

Example:

Spawn delay = 2 - (Score / 50)

Use a minimum value such as 0.5 seconds so it doesn't become impossible.

Step 7: Add polish

Ideas:
- Health bar
- Ammo and reloading
- Different zombie types
  - Fast zombies
  - Tank zombies
- Sound effects
- Survival timer
- Wave system
- Power-ups
  - Medkits
  - Speed boost
  - Shotgun

Suggested project structure:
- Stage
  - Score variable
  - Health variable
  - Game Over screen
- Player sprite
  - Movement
  - Shooting
- Zombie sprite
  - Cloning
  - Chasing
- Bullet sprite
  - Damage
  - Collision

A good first goal is: player movement → zombie chasing → health → shooting. Once those four systems work, you'll already have a playable zombie survival game.
