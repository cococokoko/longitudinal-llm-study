A good secondhand laptop for learning Python doesn't need to be expensive. Python itself runs well on modest hardware, so the best value is often a business-class laptop that's 3–7 years old.

Good options to look for:

- Lenovo ThinkPad T480
  - Intel 8th-gen Core i5 or i7
  - Easy to upgrade RAM and SSD
  - Excellent keyboard
  - Often available for $150–300 USD

- Lenovo ThinkPad T490
  - Slightly newer and lighter than the T480
  - Good Linux compatibility
  - Usually $250–400 USD

- Dell Latitude 5490 or 5400
  - Durable business laptops
  - Easy maintenance
  - Typically $180–350 USD

- HP EliteBook 840 G5 or G6
  - Solid build quality
  - Good display options
  - Usually $200–400 USD

- Dell XPS 13 (9370 or newer)
  - More premium and portable
  - Great screen
  - Often $300–500 USD used

Minimum specifications I'd recommend:
- CPU: Intel Core i5 8th generation or newer (or AMD Ryzen 5 equivalent)
- RAM: 8 GB minimum; 16 GB preferred
- Storage: 256 GB SSD minimum
- Screen: 1080p (1920×1080) if possible

If you plan to:
- Learn Python basics: 8 GB RAM is enough.
- Use VS Code, Jupyter notebooks, web development, or light data analysis: 16 GB RAM is preferable.
- Work with machine learning: start with 16 GB RAM, but many courses use cloud services anyway.

If you're comfortable installing Linux, older ThinkPads and Latitudes are especially good choices because they have excellent Linux support.
