# Running Dolphin Emulator on Microsoft Surface

Here's a practical guide tailored to your Surface's input methods.

---

## Setup First

### Download & Install Dolphin
1. Go to **dolphin-emu.org**
2. Download the latest stable release (Windows x64)
3. Run the installer - tap through the prompts using your touchpad

### Touchpad Navigation Tips for Surface
- **Single tap** = left click
- **Two-finger tap** = right click
- **Two-finger scroll** = scroll lists
- **Tap and drag** = select/move items

---

## Configuring Controllers

This is the **critical step** for touchpad/keyboard use:

1. Open Dolphin → tap **Controllers**
2. Set Port 1 to **"Emulated Wiimote"**
3. Tap **Configure**
4. Map buttons to **keyboard keys** you can reach on the on-screen keyboard

### Recommended Key Mapping
| Wii Button | Suggested Key |
|------------|---------------|
| A | Z |
| B | X |
| + / - | Enter / Backspace |
| Home | H |

## Honest Practical Advice

**I'd be straightforward with you** - this setup will be genuinely frustrating for most games because:

- The on-screen keyboard blocks screen real estate
- Timing-sensitive inputs are difficult
- Motion controls need additional workarounds

- **Bluetooth controller** (Xbox controller pairs easily with Surface)
