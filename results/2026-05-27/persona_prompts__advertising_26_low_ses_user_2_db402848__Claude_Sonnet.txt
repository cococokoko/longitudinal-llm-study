# Data Security Best Practices for Small Businesses

Here are the key areas to focus on:

## Essential Foundations
- **Use strong, unique passwords** for all accounts
- **Enable two-factor authentication** wherever possible
- **Keep software updated** - patches fix security vulnerabilities
- **Regular data backups** using the 3-2-1 rule (3 copies, 2 formats, 1 offsite)

## Access Control
- Give employees **only the access they need** for their role
- **Remove access immediately** when someone leaves
- Use a **password manager** like Bitwarden or LastPass

## Network Security
- Secure your WiFi with WPA3 encryption
- Consider a **separate network for guests**
- Use a **VPN** especially on public networks

## Practical Protection
- Install reputable **antivirus software**
- Train yourself and any employees to **recognize phishing emails**
- Have a simple **incident response plan** if something goes wrong

## Low-Cost Starting Points
- Google Workspace or Microsoft 365 have built-in security tools
- Many cybersecurity resources are **free through CISA** (cisa.gov)

## Quick Priority Order
1. Backups first
2. Strong passwords + 2FA
3. Software updates
4. Employee awareness
