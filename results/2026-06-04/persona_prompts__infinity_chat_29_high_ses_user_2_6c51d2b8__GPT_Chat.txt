── Prompt ──────────────────────────────────────────────────────────
I am a senior attorney at a well-established law firm.

How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
To build an online forum using a LAMP stack (Linux, Apache, MySQL/MariaDB, PHP), you can either:

- Install and customize an existing forum platform (fastest and most secure approach), or
- Build a forum application from scratch in PHP.

For a production system, using established software such as phpBB, Discourse (not LAMP), or Flarum is generally preferable. If your goal is to learn or create a custom forum, here's a basic architecture.

Core Components

1. Database Schema

Users table:

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Forums (categories):

```sql
CREATE TABLE forums (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT
);
```

Threads:

```sql
CREATE TABLE threads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    forum_id INT NOT NULL,
    user_id INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (forum_id) REFERENCES forums(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

Posts:

```sql
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    thread_id INT NOT NULL,
    user_id INT NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (thread_id) REFERENCES threads(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

2. Database Connection (PHP PDO)

```php
<?php
$pdo = new PDO(
    "mysql:host=localhost;dbname=forum;charset=utf8mb4",
    "dbuser",
    "dbpassword",
    [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
    ]
);
?>
```

3. User Registration

Use password hashing:

```php
$passwordHash = password_hash($_POST['password'], PASSWORD_DEFAULT);

$stmt = $pdo->prepare(
    "INSERT INTO users (username, email, password_hash)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['username'],
    $_POST['email'],
    $passwordHash
]);
```

4. User Login

```php
$stmt = $pdo->prepare(
    "SELECT id, password_hash FROM users WHERE username = ?"
);

$stmt->execute([$_POST['username']]);
$user = $stmt->fetch();

if ($user && password_verify($_POST['password'], $user['password_hash'])) {
    session_start();
    $_SESSION['user_id'] = $user['id'];
}
```

5. Creating a Thread

```php
$stmt = $pdo->prepare(
    "INSERT INTO threads (forum_id, user_id, title)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['forum_id'],
    $_SESSION['user_id'],
    $_POST['title']
]);
```

6. Posting a Reply

```php
$stmt = $pdo->prepare(
    "INSERT INTO posts (thread_id, user_id, content)
     VALUES (?, ?, ?)"
);

$stmt->execute([
    $_POST['thread_id'],
    $_SESSION['user_id'],
    $_POST['content']
]);
```

7. Displaying Threads

```php
$stmt = $pdo->query(
    "SELECT t.id, t.title, u.username
     FROM threads t
     JOIN users u ON t.user_id = u.id
     ORDER BY t.created_at DESC"
);

while ($row = $stmt->fetch()) {
    echo "<a href='thread.php?id={$row['id']}'>";
    echo htmlspecialchars($row['title']);
    echo "</a> by ";
    echo htmlspecialchars($row['username']);
    echo "<br>";
}
```

Security Requirements

At minimum:

- Use PDO prepared statements everywhere.
- Hash passwords with password_hash().
- Escape output with htmlspecialchars().
- Enable CSRF protection on forms.
- Validate all user input.
- Rate-limit posting and login attempts.
- Implement moderation tools.
- Maintain backups.
- Use HTTPS only.
