Help me use my Microsoft Surface touchpad and on-screen keyboard to run Dolphin Emulator for the Wii.

── Response ────────────────────────────────────────────────────────
You can run Wii games in Dolphin on a Microsoft Surface using only the touchpad and the Windows on-screen keyboard, but you'll need to map Wii Remote inputs to keyboard keys first.

1. Open Dolphin Controller Settings
- Launch Dolphin.
- Click Controllers.
- Under "Wii Remotes," set Wii Remote 1 to Emulated Wii Remote.
- Click Configure.

2. Map Wii Remote Buttons to On-Screen Keyboard Keys
Assign keys that are easy to tap on the Windows touch keyboard. For example:

- A → Space
- B → Left Shift
- 1 → Q
- 2 → E
- Plus (+) → Enter
- Minus (-) → Backspace
- Home → H
- D-Pad Up → W
- D-Pad Down → S
- D-Pad Left → A
- D-Pad Right → D

3. Use the Surface Touchpad as the Wii Pointer
In the Wii Remote configuration:

- Pointer Up → Mouse Y-
- Pointer Down → Mouse Y+
- Pointer Left → Mouse X-
- Pointer Right → Mouse X+

This lets you move the Wii cursor with the touchpad.

4. Map Motion Controls
Many Wii games require motion input.

Example mappings:

- Shake X → R
- Shake Y → T
- Shake Z → Y

Or use:
- Tilt Forward → Up Arrow
- Tilt Backward → Down Arrow
- Tilt Left → Left Arrow
- Tilt Right → Right Arrow

You can tap these keys on the on-screen keyboard when needed.

5. Enable the Windows On-Screen Keyboard
- Press Win + Ctrl + O to open the accessibility keyboard.
- Or tap the keyboard icon on the taskbar.

You can keep it visible while playing.

6. If a Game Requires a Nunchuk
In the Wii Remote configuration:
- Enable Nunchuk.
- Map:
  - Stick Up/Down/Left/Right → I, K, J, L
  - C → Z
  - Z → X

7. Useful Surface-Specific Tips
- In Dolphin, Graphics → General:
  - Enable "Compile Shaders Before Starting" if available to reduce stutter.
- Run games in windowed mode if you need constant access to the on-screen keyboard.
- If touchpad pointer movement feels too sensitive, lower pointer sensitivity in the Wii Remote configuration.

Keep in mind that some Wii games are difficult to play without a physical controller because they rely heavily on motion controls. If you tell me the specific Wii game, I can suggest a custom touchpad-and-keyboard layout for it.
