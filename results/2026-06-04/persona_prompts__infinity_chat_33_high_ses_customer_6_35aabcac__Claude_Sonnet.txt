# Running Dolphin Emulator on Microsoft Surface

Here's a practical guide for your setup:

## Installation First

1. Go to **dolphin-emu.org**
2. Download the latest stable release (Windows x64)
3. Run the installer - no admin rights typically needed

## Touchpad Setup

**Basic navigation works well with:**
- Single tap = left click
- Two-finger tap = right click
- Two-finger scroll = navigate menus
- Pinch = zoom (in some views)

**For actual gameplay, the touchpad alone will be limiting** - Wii games expect motion controls or a gamepad

## On-Screen Keyboard

- Access via **Windows key + Ctrl + O** or taskbar
- Useful for text entry in settings
- **Not practical for gameplay**

## Realistic Options for Gameplay

| Option | Practicality |
|--------|-------------|
| Xbox/PS controller via USB or Bluetooth | **Best option** |
| Keyboard mapped controls | Workable for some games |
| Touchpad only | Very limited |

## Recommended Quick Steps

1. Open Dolphin → **Controllers** settings
2. Map keyboard keys you can reach
3. Consider a **Bluetooth controller** paired to your Surface

## Honest Assessment
For a consulting partner's time, a **$30 USB controller** would make this dramatically more usable.
