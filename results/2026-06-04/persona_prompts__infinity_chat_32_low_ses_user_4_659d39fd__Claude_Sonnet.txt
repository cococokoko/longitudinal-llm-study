# Making a Zombie Survivor Game on Scratch

Here's a breakdown of how to build it!

## Basic Setup

**Sprites you'll need:**
- Player character
- Zombie enemy (can duplicate for multiple)
- Bullet/projectile
- Health bar or hearts
- Game Over screen

---

## Core Mechanics

### Player Movement
```
when [green flag clicked]
forever
  if <key [w] pressed?> then
    change y by (5)
  end
  if <key [s] pressed?> then
    change y by (-5)
  end
  if <key [a] pressed?> then
    change x by (-5)
  end
  if <key [d] pressed?> then
    change x by (5)
  end
end
```

### Zombie Following Player
```
when [green flag clicked]
forever
  point towards [Player]
  move (2) steps
end
```

### Shooting
```
when [space] pressed
create clone of [Bullet]
```

**Bullet behavior:**
```
when I start as a clone
point towards [mouse pointer]
forever
  move (10) steps
  if touching [Zombie] then
    broadcast [hit zombie]
    delete this clone
  end
end
```

---

## Health System

**Use a variable called `Health`**
- Set to 3 or 100 at start
- When zombie touches player, decrease health
- When health hits 0, broadcast Game Over

```
when [green flag clicked]
set [Health] to (3)
forever
  if <touching [Zombie]?> then
    change [Health] by (-1)
    wait (1) seconds
  end
  if <(Health) = (0)> then
    broadcast [Game Over]
  end
end
```

---

## Wave System

```
when [green flag clicked]
set [Wave] to (1)
forever
  wait (10) seconds
  change [Wave] by (1)
  broadcast [spawn zombies]
end
```

**Spawn more zombies each wave using clones**

---

## Tips to Polish It

- **Add a score** that increases when zombies die
- **Random spawn points** at screen edges
- **Speed up zombies** as waves increase
- **Add costumes** for walking animations
- **Sound effects** using Scratch's sound blocks
