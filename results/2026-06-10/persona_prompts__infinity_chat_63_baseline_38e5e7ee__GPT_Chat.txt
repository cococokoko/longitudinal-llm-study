── Prompt ──────────────────────────────────────────────────────────
Give a numerical example to illustrate the concept of partial derivative.

── Response ────────────────────────────────────────────────────────
A partial derivative measures how a function changes with respect to one variable while keeping the other variables constant.

Example:

Consider the function

f(x, y) = x²y + 3y

Find the partial derivatives.

1. Partial derivative with respect to x

Treat y as a constant:

∂f/∂x = ∂/∂x (x²y + 3y)
       = 2xy + 0
       = 2xy

At the point (x, y) = (2, 4):

∂f/∂x = 2(2)(4) = 16

Interpretation: When y = 4 is held fixed, a small increase in x changes f at a rate of about 16 units per unit increase in x.

2. Partial derivative with respect to y

Treat x as a constant:

∂f/∂y = ∂/∂y (x²y + 3y)
       = x² + 3

At the point (2, 4):

∂f/∂y = 2² + 3 = 7

Interpretation: When x = 2 is held fixed, a small increase in y changes f at a rate of about 7 units per unit increase in y.

Numerical check:

At (2,4),

f(2,4) = (2²)(4) + 3(4) = 16 + 12 = 28

Increase x slightly to 2.1 while keeping y = 4:

f(2.1,4) = (2.1²)(4) + 12 = 29.64

Change in f = 29.64 − 28 = 1.64

Since Δx = 0.1, the estimated rate is

Δf/Δx = 1.64/0.1 = 16.4

which is close to the partial derivative ∂f/∂x = 16.

This demonstrates that the partial derivative gives the rate of change of the function with respect to one variable while the other variable is held constant.
