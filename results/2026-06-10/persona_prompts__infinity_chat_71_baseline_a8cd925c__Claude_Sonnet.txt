## Example of an Optimization Technique in Machine Learning

**Stochastic Gradient Descent (SGD)**

This is one of the most widely used optimization techniques in machine learning. It works by:

1. **Randomly selecting** a small subset (mini-batch) of training data
2. **Computing the gradient** of the loss function
3. **Updating model parameters** in the direction that minimizes the loss

### Other Common Examples:
| Technique | Description |
|-----------|-------------|
| **Adam** | Adaptive learning rate optimizer |
| **RMSprop** | Adjusts learning rates based on recent gradients |
| **Momentum** | Accelerates gradient descent in relevant directions |
| **AdaGrad** | Adapts learning rate to parameters |

SGD and its variants are fundamental to training neural networks and other machine learning models.
