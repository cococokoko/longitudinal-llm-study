── Prompt ──────────────────────────────────────────────────────────
I work shifts at a local hospital as a patient care aide.

How do I code an online forum using LAMP?

── Response ────────────────────────────────────────────────────────
Building an online forum using the **LAMP** stack (**L**inux, **A**pache, **M**ySQL, **P**HP) is a classic, highly rewarding web development project. Since you work shift work, we will approach this in a **modular, step-by-step way** so you can code it in short bursts (30–60 minute sessions) between your shifts.

Here is a complete guide to building a simple, functional forum from scratch.

---

### Phase 1: Setting Up Your Environment (The "L-A-M-P")

Before writing code, you need a local environment on your computer to run it.

*   **Windows:** Download and install **XAMPP** or **WampServer**.
*   **Mac:** Download and install **MAMP**.
*   **Linux:** Run `sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql`.

These packages install Apache (web server), MySQL (database), and PHP (programming language) all at once. Once installed, open the control panel and click **Start** next to Apache and MySQL.

---

### Phase 2: Designing the Database (MySQL)

A forum needs to store users, categories, topics (threads), and posts (replies). 

Open **phpMyAdmin** (usually at `http://localhost/phpmyadmin`) and create a database named `hospital_forum`. Run the following SQL queries to create your tables:

```sql
-- 1. Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. Topics Table (The threads)
CREATE TABLE topics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

-- 3. Posts Table (The replies inside a thread)
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    content TEXT NOT NULL,
    topic_id INT,
    user_id INT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (topic_id) REFERENCES topics(id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

---

### Phase 3: Connecting PHP to MySQL

In your web server's root directory (usually `C:/xampp/htdocs/` or `/var/www/html/`), create a folder named `forum`.

Create a file named `db.php`. This file will connect your PHP code to your MySQL database.

```php
<?php
// db.php
$host = 'localhost';
$db   = 'hospital_forum';
$user = 'root'; // Default XAMPP user
$pass = '';     // Default XAMPP password is empty
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
     $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
     throw new \PDOException($e->getMessage(), (int)$e->getCode());
}
?>
```

### Phase 4: User Registration & Login

To post on a forum, users need to log in. We will use PHP **Sessions** to keep track of logged-in users.

#### 1. Registration (`register.php`)
```php
<?php
require 'db.php';
if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    // Securely hash the password
    $password = password_hash($_POST['password'], PASSWORD_BCRYPT);

    $stmt = $pdo->prepare('INSERT INTO users (username, password) VALUES (?, ?)');
    try {
        $stmt->execute([$username, $password]);
        header("Location: login.php");
        exit;
    } catch (Exception $e) {
        echo "Username already taken.";
    }
}
?>
<form method="POST">
    <h2>Register</h2>
    <input type="text" name="username" placeholder="Username" required><br><br>
    <input type="password" name="password" placeholder="Password" required><br><br>
    <button type="submit">Register</button>
</form>
```

#### 2. Login (`login.php`)
```php
<?php
require 'db.php';
session_start();

if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $username = $_POST['username'];
    $password = $_POST['password'];

    $stmt = $pdo->prepare('SELECT * FROM users WHERE username = ?');
    $stmt->execute([$username]);
    $user = $stmt->fetch();

    if ($user && password_verify($password, $user['password'])) {
        $_SESSION['user_id'] = $user['id'];
        $_SESSION['username'] = $user['username'];
        header("Location: index.php");
        exit;
    } else {
        echo "Invalid credentials.";
    }
}
?>
<form method="POST">
    <h2>Login</h2>
    <input type="text" name="username" placeholder="Username" required><br><br>
    <input type="password" name="password" placeholder="Password" required><br><br>
    <button type="submit">Login</button>
</form>
```

### Phase 5: The Forum Homepage (`index.php`)

This page displays all existing discussion topics and allows logged-in users to create new topics.

```php
<?php
require 'db.php';
session_start();

// Handle creating a new topic
if ($_SERVER['REQUEST_METHOD'] == 'POST' && isset($_SESSION['user_id'])) {
    $title = $_POST['title'];
    $userId = $_SESSION['user_id'];

    $stmt = $pdo->prepare('INSERT INTO topics (title, user_id) VALUES (?, ?)');
    $stmt->execute([$title, $userId]);
}

// Fetch all topics
$stmt = $pdo->query('SELECT topics.*, users.username FROM topics JOIN users ON topics.user_id = users.id ORDER BY topics.created_at DESC');
$topics = $stmt->fetchAll();
?>

<!DOCTYPE html>
<html>
<head><title>Hospital Staff Forum</title></head>
<body>
    <h1>Hospital Staff Forum</h1>
    
    <?php if (isset($_SESSION['username'])): ?>
        <p>Welcome, <?php echo htmlspecialchars($_SESSION['username']); ?>! <a href="logout.php">Logout</a></p>
        
        <h3>Create a New Topic</h3>
        <form method="POST">
            <input type="text" name="title" placeholder="Topic Title" required style="width: 300px;"><br><br>
            <button type="submit">Post Topic</button>
        </form>
    <?php else: ?>
        <p><a href="login.php">Login</a> or <a href="register.php">Register</a> to post.</p>
    <?php endif; ?>

    <h2>Discussions</h2>
    <ul>
        <?php foreach ($topics as $topic): ?>
            <li>
                <a href="topic.php?id=<?php echo $topic['id']; ?>">
                    <?php echo htmlspecialchars($topic['title']); ?>
                </a> 
                (Started by <?php echo htmlspecialchars($topic['username']); ?>)
            </li>
        <?php endforeach; ?>
    </ul>
</body>
</html>
```
