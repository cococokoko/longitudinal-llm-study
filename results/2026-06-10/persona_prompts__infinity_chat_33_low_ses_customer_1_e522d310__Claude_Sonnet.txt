# Running Dolphin Emulator on Microsoft Surface with Touch Controls

## What You'll Need
- Microsoft Surface (running Windows)
- Dolphin Emulator installed
- Wii game ROMs (that you legally own)
- Patience with touch controls

---

## Step 1: Download and Install Dolphin

1. Go to **dolphin-emu.org** in your browser
2. Tap **Download** and choose the latest stable version
3. Tap the downloaded file to install it
4. Follow the installation prompts

---

## Step 2: Basic Touchpad Navigation in Dolphin

The touchpad works like a standard mouse:

- **Single tap** = left click
- **Two-finger tap** = right click
- **Two-finger scroll** = scroll through menus
- **Tap and drag** = select/move items

---

## Step 3: Set Up the On-Screen Keyboard

**Enable the touch keyboard:**
- Tap the **keyboard icon** in your taskbar system tray
- Or go to **Settings → Accessibility → Keyboard**

**Pin it for easy access:**
- Right-click the taskbar
- Enable **Show touch keyboard button**

---

## Step 4: Configure Controllers in Dolphin

1. Open Dolphin
2. Tap **Controllers** in the top menu
3. For **Wii Remote 1**, select **Emulated Wii Remote**
4. Tap **Configure**

**Map your controls:**
- You can map keyboard keys to Wii buttons
- Tap each button field and press the corresponding key on the on-screen keyboard

**Suggested basic mapping:**
| Wii Button | Keyboard Key |
|------------|-------------|
| A | Z |
| B | X |
| + (Plus) | Enter |
| - (Minus) | Backspace |
| Home | Escape |
| D-Pad | Arrow Keys |
| 1 | Q |
| 2 | W |

---

## Step 5: Motion Controls (Wiimote Tilt/Shake)

Since you don't have a physical Wiimote:

1. In controller config, find **Tilt** and **Shake** options
2. Map these to keyboard keys
3. Alternatively, enable **Touchscreen** input if your Surface supports it

**For games requiring heavy motion controls**, consider:
- Buying a cheap Bluetooth Wiimote (connects easily to Surface)
- Using a USB gamepad instead

---

## Step 6: Loading a Game

1. Tap **Open** or add your game folder via **Config → Paths**
2. Double-tap your game to launch it
3. The game should start automatically

---

## Tips for Touch Use

- **Zoom in** on the on-screen keyboard if buttons are too small
- Use **tablet mode** in Windows for easier touch navigation
- **Pin Dolphin to taskbar** for quick access between shifts
- Enable **fullscreen** by tapping the maximize button for better gameplay

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Game runs slow | Lower graphics settings in Config → Graphics |
| Keyboard won't appear | Check Settings → Accessibility → Keyboard |
| Controller not responding | Re-check button mappings in Controllers menu |
| Game won't load | Check file format (needs .iso, .wbfs, .rvz) |

---

## Quick Access Tip
Since your schedule is unpredictable, you can **save your controller profile** in Dolphin so you don't have to reconfigure every time you sit down to play.
